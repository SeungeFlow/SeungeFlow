# brake_test_protocol.md

## Definition

Brake Test는 구조를 억지로 완성하기 위한 절차가 아니다.

Brake Test는 어디까지 구조연산이 가능하고, 어디서 source boundary가 막히며, 무엇을 guard로 남겨야 하고, 무엇을 보류해야 하는지 드러내는 절차이다.
