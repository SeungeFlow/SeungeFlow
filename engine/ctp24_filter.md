# ctp24_filter.md

## Role

Ctp24는 새 입력을 실체 / guard / field sample / 보류 / noise로 분류한다.

## Procedure

```text
1. C=(m,t,p,?) 중 어디에 들어가는지 본다.
2. R01~R12 중 어느 cell에 들어가는지 본다.
3. S₁~S₄ 중 어디에 연결되는지 본다.
4. source identity가 있는지 확인한다.
5. status를 판정한다.
```

## Output

```text
substance
guard
field_sample
pending
noise
```
