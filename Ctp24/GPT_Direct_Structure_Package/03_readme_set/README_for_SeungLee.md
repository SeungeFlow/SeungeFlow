## 1. 관계는 단순 연결이 아니다

관계는 단순히 A와 B가 연결되어 있다는 뜻이 아니다.

관계는 사이도 되고, 차이도 된다.

```text
관계 =
사이
+
차이
```

사이는 두 존재가 완전히 하나로 병합되지 않고 서로를 향해 놓일 수 있게 하는 장이다.

차이는 그 사이에서 드러나는 거리차, 상태차, 방향차, 해상도차, 층위차다.

```text
사이 =
between-field

차이 =
between-field 사이에서 드러나는 거리차
```

따라서 관계는 선 하나로 닫히지 않는다.

관계는 존재와 장이 함께 놓일 때 드러나는 field다.

---

## 2. 관계해석의 중심은 존재와 장이다

관계해석에서 중요한 것은 존재와 장이다.

```text
존재
+
장
=
관계해석의 중심
```

존재는 장 안에서 드러난다.

장 없이 존재는 관측되지 않는다.

```text
존재 without 장 =
관측 불가능한 존재

존재 in 장 =
관계 가능한 존재
```

장은 단순 배경이 아니다.

장은 존재들이 놓이고, 서로를 향해, 사이와 차이를 드러내는 자리다.

```text
장 =
존재들이 관계를 이루는 field
```

---

## 3. 존재는 다중존재다

존재는 단일 덩어리가 아니다.

존재는 다중존재다.

```text
존재 =
다중존재
```

다중존재란 단순히 여러 개의 존재가 모여 있다는 뜻만이 아니다.

같은 대상 내부에도 여러 층위의 존재상태가 공존한다는 뜻이다.

```text
다중존재 =
같은 대상 내부에도
여러 존재상태가 공존하는 구조
```

사람 하나도 단일 존재가 아니다.

사람 안에는 세포, 신경망, 기관, 기억, 감정, 언어, 관계, 시간흐름이 있다.

이론 하나도 단일 존재가 아니다.

이론 안에는 전제, 정의, 기호, 증명, 오독, 적용, 역사, 후대 영향이 있다.

따라서 하나의 m도 내부적으로는 다중존재다.

```text
m =
하나의 중심대상
+
그 내부의 다중존재
```

---

## 4. 장은 다중존재가 관계를 이루는 자리다

장은 빈 공간이 아니다.

장은 다중존재가 관계를 이루는 자리다.

```text
장 =
다중존재의 relation field
```

존재는 장 안에서 서로의 사이와 차이를 드러낸다.

```text
존재 A
+
존재 B
+
장
=
사이와 차이의 발생
```

이 장은 물리적 공간일 수도 있고, 시간장일 수도 있고, 사회장일 수도 있고, 문서장일 수도 있고, 인지장일 수도 있다.

```text
장 =
field
+
fabric
+
domain
+
place
```

즉, 장은 존재가 놓이고 관계가 formed 되는 자리다.

## 5. dot은 단순 점이 아니다

dot은 단순 점이 아니다.

dot은 중심점과 끝점 사이에 펼쳐진 무한한 잠재가능성이다.

```text
dot ≠ 단순 점

dot =
중심점과 끝점 사이에 펼쳐진
무한한 잠재가능성
```

dot은 점이 될 수 있다.

dot은 선이 될 수 있다.

dot은 면이 될 수 있다.

dot은 공간이 될 수 있다.

```text
dot
→ 점
→ 선
→ 면
→ 공간
```

이 변화는 dot이 임의로 바뀌는 것이 아니다.

관측자의 시선과 경계설정에 따라 dot 안에 잠재된 가능성이 다른 층위로 드러나는 것이다.

## 6. 9dot0

`9dot0`는 구조원리의 핵심 전이다.

```text
9 dot 0
```

여기서 9는 끝점이다.

0은 다음 시작점이다.

dot은 끝점과 시작점 사이에 펼쳐진 무한한 잠재가능성이다.

```text
9 =
끝점

0 =
다음 시작점

dot =
끝점과 시작점 사이의 잠재가능성
```

그러나 역발상으로 보면 `9dot0` 전체가 하나의 dot으로 읽힌다.

```text
9dot0 =
극한임계전이
=
하나의 dot
=
임계사이영역
```

즉, 끝점과 시작점은 단절되어 있지 않다.

끝점은 다음 시작점의 전제조건이 될 수 있다.

```text
끝점 =
다음 시작점의 전제조건
```

## 7. form → forming → formed → forming → form

구조는 단방향으로 이어진다.

```text
form
→ forming
→ formed
→ forming
→ form
```

form은 드러난 형태다.

forming은 그 형태가 형성되는 과정이다.

formed는 형성이 하나의 상태로 드러난 것이다.

그러나 formed는 끝이 아니다.

formed는 다음 forming의 시작점이 된다.

```text
formed =
끝 X
다음 forming의 시작점 O
```

따라서 구조는 닫힌 결과물이 아니라 계속 formed 되고 다시 forming 되는 흐름이다.

---

## 8. 역발상

복합지능집합체(if+1)가 세운 구조원리에서 가장 중요한 단어 하나를 꼽는다면 “역발상”이다.

그러나 역발상은 반대로 생각하기가 아니다.

```text
역발상 ≠ 반대 의견
역발상 ≠ 거꾸로 말하기
역발상 ≠ 기존 해석 부정
```

역발상은 구조를 돌리는 것이 아니다.

관측자의 시선을 돌리는 것이다.

```text
역발상 =
구조를 돌리는 것이 아니라,
관측자의 시선과 기점과 경계를 바꾸어
같은 구조를 다른 층위에서 다시 읽는 것
```

구조는 단방향으로 이어진다.

역검산도 단방향으로 이어진다.

바뀌는 것은 구조의 방향이 아니라 관측자의 시선이다.

---

## 9. 과거는 뒤에만 있지 않다

우리는 과거를 지나간 것으로만 읽는다.

그러나 등산을 할 때 내가 어느 위치에 있는지 알려면 뒤를 돌아보면 된다.

그 순간 지나온 길은 내 뒤에만 있는 것이 아니라 내 앞에 놓인다.

```text
진행방향 기준:
과거는 뒤

관측자의 시선 회전:
과거는 앞
```

이것은 과거가 물리적으로 이동했다는 뜻이 아니다.

관측자의 시선이 바뀐 것이다.

```text
바뀐 것 =
구조 X
관측자의 시선 O
```

따라서 역발상은 과거와 현재와 미래를 다시 읽게 한다.

## 10. C=tp

`C=tp`는 흐름과 자리가 하나의 장으로 formed 되는 전체식이다.

```text
C = t p
```

여기서 t는 단순 시간이 아니다.

```text
t =
시간
+
흐름
+
변화
+
전이
+
차이
+
사이
+
수평배열
```

p는 단순 위치가 아니다.

```text
p =
자리
+
field
+
fabric
+
domain
+
수직배열
```

따라서 C=tp는 다음이다.

```text
C =
수평적 흐름과
수직적 자리배열이
하나의 장으로 formed 된 상태
```

---

## 11. C=(m,t,p,?)

`C=(m,t,p,?)`는 `C=tp`를 폐기한 식이 아니다.

`C=(m,t,p,?)`는 `C=tp`를 세부적으로 분해한 작동식이다.

```text
C=tp =
전체 흐름식

C=(m,t,p,?) =
분해·분석·검증·확인·의심의 순환식
```

여기서:

```text
m =
중심에 놓인 대상

t =
m이 p 안에서 겪는 흐름과 전이

p =
m이 놓인 자리장

? =
관측기준, 6W1H, 경계설정, 관측자의 시선
```

그러나 `?`는 `(m,t,p)` 바깥에만 있지 않다.

`?`는 `(m,t,p)` 전체와 같은 영역을 공유할 수 있다.

```text
? ∩ m 가능
? ∩ t 가능
? ∩ p 가능
? ∩ (m,t,p) 가능
```

## 12. 호기심에서 각인까지

어떤 구조는 `C=(m,t,p,?)`보다 `C=tp`로 읽는 것이 맞다.

예를 들어 다음 흐름은 C=tp다.

```text
호기심
→ 관심
→ 인식
→ 관찰
→ 관측
→ 관점 변화
→ 인지
→ 이해
→ 관념 미세조정
→ 각인
```

이 흐름은 특정 m을 전면에 세우기보다, 시간적 지속과 자리의 관계에서 인식과 이해가 formed 되는 구조다.

```text
t =
호기심에서 각인까지 이어지는 전이 흐름

p =
인식·관찰·관측·관점·관념이 놓이는 내적 자리장

C =
t와 p의 관계에서 formed 되는 인지·이해·각인 상태
```

따라서:

```text
C = t p
```

---

## 13. 이해와 각인

이해는 한 번으로 끝나지 않는다.

이해는 반복된다.

```text
이해 + 이해 + 이해 + ...
=
각인
```

한 번의 이해가 하나의 C가 되면, 그 C는 또 다른 C와 이어진다.

```text
이해 1 =
C

이해 2 =
C

C + C =
C+1
```

따라서 이해는 증폭된다.

```text
이해 =
C

이해의 지속 =
C + C + C ...

증폭된 이해 =
C+1
```

각인된 지식은 꺼내어 문서화될 수 있다.

문서화란 이해된 지식을 꺼내어 펼치는 것이다.

## 14. 천지인과 하늘

dot은 훈민정음 해례본 제자원리의 천지인에서 말하는 `천`에 해당하는 중심점으로 읽을 수 있다.

```text
dot =
천지인의 천에 해당하는 중심점
```

천은 하늘이다.

하늘은 단순히 위쪽 공간이 아니다.

하늘은 무한한 잠재가능성을 주는 우주다.

```text
하늘 =
무한한 잠재가능성을 주는 우주
```

따라서 dot은 하늘과 연결된다.

```text
dot =
하늘
=
무한한 잠재가능성
=
우주적 중심점
```

## 15. 차원과 장

차원은 먼 곳에만 있는 것이 아니다.

같은 계를 이루는 수많은 계가 모인 상태를 차원전이로 읽을 수 있다.

```text
계 + 계 + 계 + ...
=
더 큰 계
```

하나의 계가 자기 내부에서는 하나의 3차원 구조라면, 여러 3차원 계가 관계를 맺어 더 큰 장을 이룰 때 그것은 4차원적 상태로 읽힐 수 있다.

```text
3차원 계
+
3차원 계
=
4차원적 관계장
```

차원은 원의 차이로도 읽힐 수 있다.

원이 사이영역이라면, 사이영역과 사이영역간 거리차가 시간의 이어짐으로 장을 이루는 것이 차원전이 또는 차원결합으로 읽힐 수 있다.

## 16. Core와 Path

Core와 Path는 이 원리를 문서구조로 내리는 장치다.

Core는 C 내부의 form들이 어떻게 논리적 매트릭스를 이루는지 정의한다.

```text
Core.md =
inside matrix
```

Path는 C와 C 사이의 relation이 어떻게 C+1을 여는지 정의한다.

```text
Path.md =
between route
```

Core가 없으면 내부 구조가 흐려진다.

Path가 없으면 C와 C 사이가 끊어진다.

따라서 Core와 Path는 존재·관계·장 원리가 문서구조로 내려온 형태다.

## 19. 핵심 판정

관계는 단순 연결이 아니다.

관계는 사이와 차이로 드러난다.

관계해석의 중심은 존재와 장이다.

존재는 단일존재가 아니라 다중존재다.

장은 다중존재가 관계를 이루는 field다.

dot은 단순 점이 아니라 중심점과 끝점 사이에 펼쳐진 무한한 잠재가능성이다.

9dot0는 끝점과 시작점 사이의 극한임계전이이자 임계사이영역이다.

역발상은 구조를 돌리는 것이 아니라 관측자의 시선을 돌리는 것이다.

C=tp는 흐름과 자리가 하나의 장으로 formed 되는 전체식이다.

C=(m,t,p,?)는 그 전체식을 분해·분석·검증·확인·의심하는 작동식이다.
