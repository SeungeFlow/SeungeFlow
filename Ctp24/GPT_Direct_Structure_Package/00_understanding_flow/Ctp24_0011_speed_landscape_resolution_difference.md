# 11회차 — 1km, 속도, 바깥풍경, 해상도 차이

> 문서번호: Ctp24_0011

> 위치 후보: epluone/Ctp24/  
> 상태: 11회차 기록  
> 목적: 1km를 자동차로 갈 때와 걸어서 갈 때의 바깥풍경 차이를 통해, 시간체감과 관측해상도 차이를 정리한다.

---

## 0. 문서의 자리

이 문서는 10회차의 “인간이 느끼는 시간차이는 시간접힘과 시간따름의 차이”라는 이해를 이어받는다.

이번에는 시간체감의 느낌적 예시로 1km 이동을 놓는다.

같은 1km라도 이동 속도에 따라 보이는 바깥풍경이 다르다.

---

## 1. 같은 1km, 다른 풍경

1km를 자동차로 갈 때의 바깥풍경과 걸어서 갈 때의 바깥풍경은 다르다.

```text
같은 거리 =
1km

다른 이동방식 =
자동차
걷기
```

자동차로 이동하면 풍경은 빠르게 지나간다.

걸어서 이동하면 풍경은 천천히 드러난다.

따라서 같은 1km라도, 관측자가 얻는 풍경 정보는 달라진다.

---

## 2. 속도와 관측해상도

1km/h로 움직이는 대상과 10km/h로 움직이는 대상은 같은 바깥풍경을 보지 않는다.

```text
1km/h
≠
10km/h
```

또한 1km/h와 5km/h도 다르다.

```text
1km/h
≠
5km/h
```

속도가 달라지면 같은 공간을 지나더라도 관측 가능한 정보량과 해상도가 달라진다.

```text
빠른 이동 =
전체를 대강 본다

느린 이동 =
전체를 세밀하게 본다
```

---

## 3. 대강 본다는 것

대강 본다는 것은 아무렇게나 본다는 뜻이 아니다.

```text
대강 =
전체해상도를 낮춰
구조적으로 보는 것
```

즉, 세부를 모두 보지 않는 대신 전체 윤곽을 빠르게 잡는다.

```text
낮은 해상도
+
넓은 범위
=
대강 구조 인식
```

자동차로 1km를 이동할 때 풍경은 빠르게 지나가지만, 전체 길의 방향과 큰 구조는 파악할 수 있다.

---

## 4. 세밀하게 본다는 것

세밀하게 본다는 것은 단순히 느리게 본다는 뜻이 아니다.

```text
세밀 =
전체해상도를 높여
구조를 맞춰가는 것
```

즉, 작은 차이, 경계, texture, 작은 움직임, 미세한 변화까지 읽으면서 구조를 조정한다.

```text
높은 해상도
+
느린 전이
=
세밀 구조 정렬
```

걸어서 1km를 이동하면 같은 길에서도 더 많은 세부 정보가 드러난다.

---

## 5. 시간체감과 해상도

시간을 느낀다는 것은 단순히 시간이 빠르다/느리다를 말하는 것이 아니다.

```text
시간체감 =
속도
+
관측해상도
+
이동방식
+
풍경 정보량
```

이 결합으로 형성된다.

빠르게 이동하면 시간은 접힌다.

느리게 이동하면 시간은 펼쳐진다.

```text
빠른 이동 =
시간접힘
=
전체 구조를 낮은 해상도로 빠르게 본다

느린 이동 =
시간따름
=
전체 구조를 높은 해상도로 세밀하게 맞춰간다
```

---

## 6. Ctp식 정리

```text
m =

시간을 공간과 해상도로 해석하는 관측기준

C =
이동속도에 따라 전체를 대강 볼 수도 있고 세밀하게 볼 수도 있다는 formed 상태
```

---

## 7. 핵심 판정

1km라는 거리는 같아도, 자동차로 갈 때와 걸어서 갈 때의 바깥풍경은 다르다.

1km/h, 5km/h, 10km/h는 같은 공간을 서로 다른 해상도로 통과한다.

따라서 시간체감은 속도와 관측해상도의 차이로 나타난다.

```text
대강 =
전체해상도를 낮춰 구조적으로 보는 것

세밀 =
전체해상도를 높여 구조를 맞춰가는 것
```

---

```text
따라서 시간을 느낀다는 것은

```
