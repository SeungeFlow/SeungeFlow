# 12회차 — 관측은 거시, 관찰은 미시

> 문서번호: Ctp24_0012  
> 파일명: macro_observation_micro_observing.md  
> 위치 후보: epluone/Ctp24/  
> 상태: 12회차 기록  
> 목적: 지식을 모으는 두 방식, 즉 100년 단위의 거시 관측과 특이점 발생 지점의 1년 단위 미시 관찰을 구분한다.

---

## 0. 문서의 자리

이 문서는 11회차의 “같은 1km라도 이동속도에 따라 전체해상도가 달라진다”는 이해를 이어받는다.

이번에는 지식을 모으는 방식 자체를 두 층으로 나눈다.

```text
관측 =
거시

관찰 =
미시
```

---

## 1. 지식을 모으는 두 가지 방식

우리가 지식을 모으는 방식은 두 가지다.

```text
1. 100년 단위로 끊어서 구조적으로 특이점을 관측하는 것

2. 특이점이 생긴 곳을 1년 단위로 관찰하는 것
```

즉, 먼저 넓게 본다.

그 다음 특이점이 생긴 지점을 좁혀서 본다.

---

## 2. 관측은 거시

관측은 넓은 시간장을 대상으로 한다.

```text
관측 =
100년 단위
=
거시적 시간면
```

100년 단위로 끊어 보면, 개별 사건 하나하나보다 구조적 특이점이 보인다.

```text
100년 단위 관측 =
시대의 구조적 변화
문명의 전환
기술의 등장
이론의 형성
언어와 사회장의 변화
```

즉, 관측은 전체 구조를 보기 위한 것이다.

```text
관측 =
전체장을 낮은 해상도로 넓게 보는 것
```

---

## 3. 관찰은 미시

관찰은 특이점이 생긴 곳을 좁혀서 본다.

```text
관찰 =
1년 단위
=
미시적 세부 확인
```

100년 단위 관측에서 특이점이 발견되면, 그 지점을 1년 단위로 다시 본다.

```text
특이점 발생 지점
→ 1년 단위 관찰
```

이때 관찰은 세부를 본다.

```text
관찰 =
전체장을 높은 해상도로 좁게 보는 것
```

---

## 4. 관측과 관찰의 차이

관측과 관찰은 같은 말이 아니다.

```text
관측 =
거시
넓은 범위
낮은 해상도
구조적 특이점 탐지

관찰 =
미시
좁은 범위
높은 해상도
특이점 내부의 세부 변화 확인
```

즉:

```text
관측은 찾는다.
관찰은 들여다본다.
```

---

## 5. 100년과 1년

100년 단위는 면을 본다.

```text
100년 =
10×10 시간면
```

1년 단위는 그 면 안의 한 지점을 본다.

```text
1년 =
365일 선
=
1궤도
```

따라서:

```text
100년 관측 =
시간면에서 특이점 찾기

1년 관찰 =
특이점이 생긴 궤도 내부를 세밀하게 보기
```

## 7. 핵심 판정

지식을 모으는 방식은 두 가지다.

```text

=
관측

특이점이 생긴 곳을 1년 단위로 들여다보는 것
=
관찰
```

따라서:

```text
관측 =
거시

관찰 =
미시
```
