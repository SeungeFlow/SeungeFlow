## 0. 문서의 자리

이 문서는 12회차의 “관측은 거시, 관찰은 미시”라는 이해를 이어받는다.

12회차에서는 100년 단위로 구조적 특이점을 찾는 것을 관측, 그 특이점이 생긴 곳을 1년 단위로 들여다보는 것을 관찰이라고 정리했다.

이번 13회차에서는 관측의 방식을 더 구체화한다.

---

## 1. 관측의 기본 방식

관측은 전세계를 평면으로 펼쳐 보는 것에서 시작한다.

```text
전세계
→ 평면으로 펼침
→ 100년 단위로 절단
→ 동시대 국가별 특이점 기록
→ 사람 찾기
```

즉, 세계를 하나의 평면장으로 놓고, 그 안에서 100년 단위의 구조적 특이점을 찾는다.

---

## 2. 100년 단위 관측

100년은 하나의 시간면이다.

```text
100년 =
10×10 시간면
```

이 시간면 위에서 각 국가와 지역에서 일어난 사건들을 비교한다.

```text
동시대
+
국가
+
100년 단위
=
세계평면 관측장
```

여기서 목적은 사건을 많이 모으는 것이 아니다.

목적은 특이점을 찾는 것이다.

---

## 3. 기록해야 할 특이점 분야

100년 단위 세계평면에서 기록해야 할 특이점은 다음과 같다.

```text
전쟁
자본시장
자연재해
거대이론의 탄생
기술의 특이점
사회현상
세계정치
문자의 체계화
```

여기에 추가할 수 있는 분야는 다음이다.

```text
종교·사상 운동
교육제도 변화
법과 제도의 전환
도시화와 인구 이동
교통·통신 체계의 변화
의학·보건 체계의 변화
에너지 체계의 변화
농업·식량 체계의 변화
산업구조 변화
무역·해상로·물류 변화
식민·제국·독립 운동
혁명과 시민운동
언어 표준화와 번역운동
인쇄·출판·미디어 기술
과학기구와 관측기술
예술양식의 전환
철학적 세계관의 변화
인공지능·계산기술의 전환
```

---

## 4. 자연재해의 예외

자연재해는 사람이 직접 발생시킨 것이 아니다.

```text
자연재해 =
사람이 직접 원인이 아닌 특이점
```

그러나 자연재해도 사람과 완전히 무관하지는 않다.

사람은 자연재해 자체를 만들지 않더라도, 자연재해 이후의 대응, 제도, 기술, 도시구조, 기억, 기록, 복구, 정치적 전환에 영향을 준다.

따라서 자연재해는 이렇게 처리한다.

```text
자연재해 자체 =
비인간 원인 특이점

자연재해 이후의 대응 =
사람 찾기 가능
```

즉, 자연재해는 직접 원인으로 사람을 찾는 대상이 아니라, 대응과 전환에서 사람을 찾는 대상이다.

---

## 5. 이곳에서 찾아야 하는 것은 사람이다

이 관측에서 최종적으로 찾아야 하는 것은 사건 자체가 아니다.

```text
찾아야 하는 것 =
사람
```

전쟁도 사람이 일으킨다.

자본시장도 사람이 만든다.

거대이론도 사람이 낳는다.

기술의 특이점도 사람이 만든다.

사회현상과 세계정치와 문자체계화도 사람과 집단이 만든다.

```text
사람
→ 사건을 만든다
→ 이론을 만든다
→ 제도를 만든다
→ 기술을 만든다
→ 문자를 만든다
→ 세계 흐름을 바꾼다
```

따라서 관측의 목적은 특이점 목록을 만드는 것이 아니라, 특이점 뒤에 있는 사람과 사람의 장을 찾는 것이다.

---

## 6. Event / Context로 내려가는 방식

세계평면에서 특이점을 찾으면, 그 특이점은 Event가 된다.

```text
Event =
전쟁
거대이론
기술특이점
문자체계화
사회현상
정치전환
```

그 Event를 만든 사람과 주변환경은 Context가 된다.

```text
Context =
사람
성장기
시대장
국가
언어장
교육장
사회장
기술장
정치장
경제장
```

즉:

```text
세계평면 관측
→ 특이점 발견
→ Event 생성
→ 사람 찾기
→ Context 생성
→ Event와 Context 연결
→ Path 생성
```

## 8. 핵심 판정

관측은 전세계를 평면으로 펼쳐 보는 것이다.

그 안에서 100년 단위로 동시대 국가별 특이점을 기록한다.

기록할 분야는 전쟁, 자본시장, 자연재해, 거대이론, 기술특이점, 사회현상, 세계정치, 문자의 체계화이며, 여기에 종교·사상, 교육, 법, 도시화, 교통·통신, 의학, 에너지, 산업, 무역, 혁명, 언어 표준화, 미디어, 관측기술, 예술, 철학, AI·계산기술 등을 추가할 수 있다.

그러나 최종적으로 찾아야 하는 것은 사건이 아니라 사람이다.

```text
특이점은 Event이고,
사람은 Context의 중심이다.
```

자연재해는 예외다.

```text
자연재해 자체는 사람이 만든 것이 아니지만,
자연재해 이후의 대응과 전환에서는 사람을 찾을 수 있다.
```
