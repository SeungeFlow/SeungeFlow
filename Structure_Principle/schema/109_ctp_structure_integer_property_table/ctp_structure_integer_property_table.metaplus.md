## 4. integer_property_detail

### 4.1 0 = 정중심

```text
0 =
정중심
```

0은 “없음”만이 아니다.

0은 중심이다.

```text
0 =
origin
center
equilibrium point
정중심 평형점
```

0은 구조가 시작되거나 균형을 잡는 중심 자리다.

### 4.2 1 = 차이

```text
1 =
차이
```

1은 단순히 하나라는 수량이 아니라, 차이가 처음 드러나는 자리다.

```text
dot 1개 =
하나

dot 2개 이상 =
사이 / 차이 발생
```

즉 1은 차이의 표시다.

```text
1 =
difference marker
```

### 4.3 2 = 평면

```text
2 =
평면
```

2는 두 방향 / 두 relation이 생기며 평면성이 열리는 자리다.

```text
xㆍdx =
평면
```

2는 단순한 수량 둘이 아니라, 평면 phase의 발생이다.

### 4.4 3 = 공간

```text
3 =
공간
```

3은 평면 밖의 방향이 열리며 공간이 되는 자리다.

```text
xㆍdxㆍddx =
공간
```

3은 3D 공간 phase의 자리성질값이다.

### 4.5 4 = 이동

```text
4 =
이동
```

4는 구조가 자리에서 다른 자리로 이동하는 성질값이다.

```text
위치상태 전이
movement
```

4는 위치가 달라지는 state transition으로 읽힌다.

### 4.6 5 = 변위

```text
5 =
변위
```

5는 단순 이동이 아니라 이동 전후의 차이값이다.

```text
이동 전 위치
+
이동 후 위치
=
변위
```

5는 displacement / difference after movement로 읽힌다.

### 4.7 6 = reset

```text
6 =
reset
```

6은 구조가 다시 기준을 잡거나, 이전 흐름을 재설정하는 자리다.

```text
reset
re-reference
다시 기준잡기
```

6은 기준 재설정의 성질값이다.

### 4.8 7 = 안정

```text
7 =
안정
```

7은 구조가 일정한 안정 상태에 들어간 자리다.

```text
stable state
formed stability
```

7은 reset 이후 형성된 안정성으로 읽을 수 있다.

### 4.9 8 = 포화

```text
8 =
포화
```

8은 내부가 충분히 차거나, 구조가 더 이상 단순 증가로 해석되지 않는 포화 상태다.

```text
saturation
fullness
포화된 상태
```

8은 단순 증가가 아니라 포화된 배치 / 상태값이다.

### 4.10 9 = 문턱

```text
9 =
문턱
```

9는 끝이 아니라 다음 전이를 앞둔 경계값이다.

```text
threshold
boundary-before-transition
전이 직전
```

9는 다음 phase로 넘어가기 전의 문턱이다.

### 4.11 10 = 9 + 1dx

```text
10 =
9 + 1dx
```

10은 단순히 십이 아니다.

9라는 문턱에 1dx가 더해져 다음 방향축으로 넘어가는 상태다.

```text
10 =
문턱 9
+
차이 1
+
방향전이 dx
```

즉:

```text
10 =
다음 축으로 넘어가는 전이값
```
