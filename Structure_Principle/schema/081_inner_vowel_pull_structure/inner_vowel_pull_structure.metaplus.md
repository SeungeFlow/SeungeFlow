## 3. source_meta_understanding

081_inner_vowel_pull_structure의 구조 이해는 다음으로 정리된다.

```text
inner_vowel_pull_structure =
inner vowel pull structure
ㆍ as pull point / direction point
ㅗ / ㅓ as inner-reference vowels
input_order → pull_direction
000dot 안의 direction sound 후보
```

### core

```text
ㆍ =
끌림점 / 방향점

ㅗ =
ㆍㅡ

ㅓ =
ㆍㅣ

안쪽 기준 =
ㅗ와 ㅓ
```

### definition

```text
ㆍ은 ㅣ 또는 ㅡ에 붙는 수동 점이 아니다.

ㆍ은 ㅣ 또는 ㅡ을 끌어당기는 방향점으로 읽을 수 있다.

ㆍ이 먼저 놓이고 그 뒤에 ㅡ 또는 ㅣ이 오면,
ㅗ와 ㅓ가 형성된다.
```

### pull_structure

```text
ㆍㅡ = ㅗ

ㆍㅣ = ㅓ
```

### 판정

```text
이 구조에서 ㆍ이 먼저 놓이고 축이 뒤따라온다.

따라서 ㅗ와 ㅓ는 안쪽 기준으로 읽을 수 있다.
```

