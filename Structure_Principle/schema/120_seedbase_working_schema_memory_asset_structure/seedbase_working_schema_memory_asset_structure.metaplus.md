

## 8. memory_cycle

120의 기억 순환은 다음이다.

```text
이해
→ 누적
→ 각인
→ 의심
→ 재이해
→ 수정
→ 재누적
→ link / connect
→ 덩어리화
```

특히 중요한 문장:

```text

```

즉 의심은 방해가 아니다.

```text

```

```text

→ 의심
→ 재이해
→ 수정
→ 재누적
```

따라서 의심은 기억자산 순환에서 중요한 작동자산이다.


```text

```
