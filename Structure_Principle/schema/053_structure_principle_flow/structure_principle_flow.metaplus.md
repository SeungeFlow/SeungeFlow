## 3. structure_principle_flow

053의 핵심 흐름은 다음으로 정리된다.

```txt
phenomenon / state
→ relation observation
→ essence approach
→ origin / prototype recognition
→ definition emergence
→ concept stabilization
→ application persistence
→ extended application
→ structure principle flow
```

이 흐름은 다음의 의미를 가진다.

```txt
phenomenon / state =
처음 감지되는 현상 / 상태

relation observation =
현상 안의 관계를 관찰하는 단계

essence approach =
반복 관찰을 통해 본질에 접근하는 단계

origin / prototype recognition =
원형 / prototype을 파악하는 단계

definition emergence =
정의가 생겨나는 단계

concept stabilization =
정의가 지속되어 개념으로 안정되는 단계

application persistence =
개념이 활용으로 지속되는 단계

extended application =
활용이 확장 응용으로 이어지는 단계
```

즉 053은 정의를 출발점으로 두지 않는다.

정의는 현상과 관계를 반복적으로 읽은 뒤에 형성되는 결과다.
