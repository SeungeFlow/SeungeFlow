## 10. x_dx_ddx_and_3d

2D에서 3D로 전이할 때 높이 개념이 발생한다.

```text
2D =
평면 =
xㆍdx

3D =
공간 =
xㆍdxㆍddx
```

여기서 ddx는 평면 밖으로 열린 direction이다.

```text
ddx =
높이 =
평면 밖 direction
```

102 기준으로 이것은 phase boundary다.

```text
2D phase:
xㆍdx 안에서 구조가 닫힘

3D phase:
xㆍdx로 닫힌 평면 밖에 ddx 방향이 열림
```

### 10.1 (x,dx)와 (x,dx,ddx)

```text
(x,dx)는
3D 안에서 2D 평면을 표현하는 구조다.
```

예:

```text
(x,y,z)에서 (x,dx) =
(x,y) =
(1,1,0)
```

즉 z가 0일 때는 아직 높이축이 열린 것이 아니다.

```text
dot이 z축을 만나야 ddx가 발생한다.
```

따라서:

```text
평면구조
→ z축과 만남
→ 높이 발생
→ 공간구조
→ 입체구조
```
