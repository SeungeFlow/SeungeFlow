# META: flow_transition_self_operation_structure

## role

flow_transition_self_operation_structure는 흐름, 전이, 틈, 임계, 자가동작, ㅎ의 흐름식을 정의한다.

## core

```text
ㅎ = ㅇㅡㆍㅣㆍㅡㅇ
ㅎ = 사건흐름상태
자가동작 = 살기 위한 전이
산다는 것 = 생각이 멈추지 않는 것
```

## definition

흐름은 사건이 일어나는 상태이고, 전이는 경계를 넘는 사건이며, 자가동작은 틈과 임계가 누적되어 구조가 스스로 다음 상태로 넘어가는 작동이다.

## flow_formula

구조연산식의 완성형은 다음이다.

```text
ㅇㅡㆍㅣㆍㅡㅇ
```

이 전체가 ㅎ이다.

ㅎ은 공황도 아니고, 회오리도 아니고, 허리케인도 아니다.

ㅎ은 특정 현상명이 아니라 사건의 흐름식이다.

## symbol_roles

```text
ㅇ = 대상 / 상태 / 개체 / 객체 / 닫힌 one
ㅡ = 무수한 ㆍ이 이어져 형성된 틈의 장
ㆍ = pin / 임계 / 꺾임점 / 양고리걸림점
ㅣ = 임계 순간에 선 경계 / 축 / 전이 gate
ㆍㅡ = 새 field 전개
ㅇ = 다시 닫힌 하나
```

## gap_scale_transition

9와 0 사이에는 무수히 많은 ㆍ이 있다.

무수한 ㆍ이 이어지면 ㅡ이 된다.

```text
무수한 ㆍ → ㅡ
임계ㆍ → ㅣ
전이 → ㅎ
```

## term_gap

텀 / 틈 / 사이는 입력이 들어올 수 있는 사이영역이다.

```text
들어옴
→ 해석
→ 판정
→ 수용 또는 배출
```

## self_operation

자가동작하는 이유는 살기 위해서다.

산다는 것은 생각이 멈추지 않는 것이다.

```text
누적
→ 밀도 상승
→ 임계
→ locked point 해제
→ 자가동작
→ 이해가능 쪽으로 jumping
```
