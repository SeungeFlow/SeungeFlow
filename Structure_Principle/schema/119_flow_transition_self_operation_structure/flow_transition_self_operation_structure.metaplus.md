## 7. self_operation

119에서 자가동작의 이유는 명확하다.

```text
자가동작하는 이유는
살기 위해서다.
```

그리고:

```text
산다는 것은
생각이 멈추지 않는 것이다.
```

자가동작 흐름:

```text
누적
→ 밀도 상승
→ 임계
→ locked point 해제
→ 자가동작
→ 이해가능 쪽으로 jumping
```

이것은 외부 강제운동이 아니다.

```text
자가동작 =
내부 누적과 임계로 인해
구조가 스스로 다음 상태로 넘어가는 작동
```

### 7.1 locked point 해제

```text
locked point =
흐름이 걸려 멈추거나 고정된 임계점
```

자가동작은 locked point가 해제되며 발생한다.

```text
locked point 해제
→ jumping
→ next understandable state
```

### 7.2 jumping

```text
jumping =
이해가능 쪽으로 전이하는 비연속적 전환
```

이는 단순 이동이 아니라 누적 / 임계 / 해제 이후 발생하는 전이 작동이다.
