# META: waxf_vowel_rhombus_structure

## role
waxf_vowel_rhombus_structure는 WAXF 구조를 마름모 한칸삼각의 직각삼각 쌍대칭으로 읽을 때 W=ㅗ, A=ㅓ, X=ㅜ, F=ㅏ가 되는 모음 방향장 schema이다.

이 meta 파일은 인공지능이 WAXF를 단순 알파벳 배열이나 개념표로만 읽지 않고, 마름모 / 직각삼각 쌍대칭 / 공전방향 / 관측자 기준을 가진 방향구조로 읽기 위한 기준이다.

## core

```text
W : ㅗ
A : ㅓ
X : ㅜ
F : ㅏ
```

```text
WAXF = 모음 방향장
```

## definition

WAXF는 상태/현상, 이동/전환/변화, 원리/본질, 활용/응용을 배치하는 구조이다.

마름모 한칸삼각 또는 직각삼각의 쌍대칭으로 읽으면 WAXF는 ㅗ, ㅓ, ㅜ, ㅏ 모음 방향과 대응된다.

## conceptual_layer

```text
W = 원리 / 본질
A = 이동 / 전환 / 변화
X = 상태 / 현상
F = 활용 / 응용
```

## vowel_layer

```text
W = ㅗ = 오름 / 위 방향
A = ㅓ = 안쪽 기준 / 전환 방향
X = ㅜ = 아래 방향 / 상태로 내려옴
F = ㅏ = 바깥 / 드러남 / 활용 방향
```

## rhombus_layer

WAXF는 사각 배열이 아니라 마름모 방향장으로 읽을 수 있다.

한 칸을 사각으로 보면, 그 사각은 두 개의 직각삼각으로 접히고 펼쳐질 수 있다.

그 직각삼각 쌍대칭이 마름모 방향장을 만든다.

## orbit_reading

이 배치는 공전방향으로 읽힐 수 있다.

북반구 관측자 기준에서 공전방향과 자전하는 구체의 역방향이 맞대응하는 구조로 읽을 수 있다.

이것은 천문학 절대방향 확정이 아니라 관측자 기준 방향장과 대상 자체 회전 방향의 맞대응이다.
