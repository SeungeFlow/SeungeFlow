# META: pin_dot_y_branch_return_structure

## role

pin_dot_y_branch_return_structure는 pin, dot, Y가지, NO 판정, 복귀, shelter, alive_like를 하나의 선택·복귀 구조로 정의한다.

## core

```text
pin = 작동 가능한 dot
Y가지 = branch 선택·복귀 구조
NO = pin으로 복귀
전부 NO = 멈춤 / 무응답 / 길 없음 선언 / shelter 복귀
```

## definition

pin은 미시세계에서는 고정핀이고, one의 층위에서는 중심점이며, 칸 구조에서는 대각 교차점이 될 수 있다.

Y가지는 pin에서 branch가 열리고, 한 branch에서 NO가 오면 다시 pin으로 되돌아오는 선택·복귀 구조다.

## pin_one_scale

```text
pin = 미시세계의 고정핀
one = 거시세계의 완전 구조체
dot = 최소자리이자 전체압축점
vectorㆍdot = 미시와 거시 양쪽에서 작동하는 연결층
```

## y_branch_return

Y가지 탐색:

```text
pin
→ branch A
→ NO
→ pin 복귀
→ branch B
→ NO
→ pin 복귀
```

모든 branch가 NO라면:

```text
멈춘다
답하지 않는다
갈 수 있는 길이 없다고 선언한다
pending으로 둔다
지나온 Y가지 shelter로 복귀한다
```

## past_y_branch

지나온 Y가지는 검증된 기억자산이다.

```text
지나온 Y가지 = alive_like shelter
```

허름하고 비가 새는 shelter라도, 되돌아갈 수 있는 곳이 없는 것보다 훨씬 낫다.

## six_direction_structure

Y가지는 평면의 두 갈래만이 아니다.

구형구조체 안에서는 한 pin에서 6방향으로 펼쳐지는 선택구조다.

```text
+X / -X
+Y / -Y
+Z / -Z
```

## branch_bridge_inclusion

하나의 존재가 무리로 들어오려면 가지 혹은 다리가 필요하다.

```text
가지 = 접촉 시도
다리 = 수용된 연결
```

## forbidden

- Y가지를 고정된 문자 모양으로 보지 않는다.
- 신경회로 방향을 하나로 고정하지 않는다.
- NO를 실패로 보지 않는다.
- 되돌아감을 패배로 보지 않는다.
- 지나온 Y가지를 폐기물로 보지 않는다.
- 신경과학 대응을 검증 없이 확정하지 않는다.

## pending

- Y가지와 신경회로 pruning/synaptogenesis 대응은 과학자료 기준으로 별도 검산한다.
- 정자 꼬리 대응예시는 생물학 사실과 구조비유를 분리해 보존한다.
- 6방향 구조는 sphere 내부 branch network와 relation으로 둔다.
