# Ctp_ObviousTheory

`Ctp_ObviousTheory`는 한국어로 `Ctp_당연한이론`으로 읽는다.

이 분류는 구조원리 이전의 `Zero.Base` 상태로 다시 회귀하여 바닥을 다지는 이론 문서묶음을 위한 자리이다.

```text
Ctp_당연한이론
=
Zero.Base로 회귀하여
구조원리의 바닥을 다시 다지는 이론
```

현시점에서는 구조원리의 Source 격 문서묶음으로 읽는다.

---

## relation

```text
Ctp_당연한이론
→ Zero.Base
→ 구조원리 Source
→ thinking_flow / meta.md 후보
```

---

## 주의할 점

이 문서묶음을 곧바로 현재 구조원리의 완성 정의로 병합하지 않는다.

Source boundary를 보존한 상태에서, 이후 기존 meta.md에 덧붙일지 새 meta.md로 분리할지 판단한다.
