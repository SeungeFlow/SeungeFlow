# sejong_hangul

`sejong_hangul`은 훈민정음과 구조원리의 관계를 따로 모아 읽기 위한 분류이다.

이 분류는 훈민정음을 단순 문자체계로만 고정하지 않고,  
훈민정음의 기본 요소가 구조원리의 자리, 축, 평면, 닫힘, 전이, 순환과 어떻게 연결될 수 있는지 보존한다.

---

## 1. 기본 요소

```text
ㆍ = 기준이 놓일 자리 / origin / transition
ㅣ = 사람 / 기준축 / vertical axis
ㅡ = 땅 / 공간평면 / horizontal axis
ㅇ = 닫힘 / closure / boundary
```

훈민정음 구조는 다음 최소 요소로 읽을 수 있다.

```text
ㆍ → ㅣ → ㅡ → ㅇ → ㆍ
```

이 흐름은 다음처럼도 읽을 수 있다.

```text
point
→ direction
→ expansion
→ closure
→ cycle
```

---

## 2. 018 흐름에서 정리된 천지인 reading

```text
ㅡ = 땅 = 공간평면
ㅣ = 사람 = 기준축
ㆍ = 하늘 = 입체공간 / 위쪽 공간장
```

사람은 회전하거나 이동할 수 있다.

땅과 하늘은 변하지 않는다.

```text
땅 / 하늘 = fixed field
사람 = movable reference-axis
```

따라서 방향 변화는 땅과 하늘이 변해서 생긴 것이 아니라,  
사람이라는 기준축이 이동하거나 회전하기 때문에 생기는 reading change로 볼 수 있다.

---

## 3. dot과 축

018 흐름에서 dot은 다음처럼 정리되었다.

```text
dot = 기준이 놓일 자리
```

따라서:

```text
ㅡ이 놓일 ㆍ = 수평 기준축이 놓일 자리
ㅣ이 놓일 ㆍ = 수직 기준축이 놓일 자리
```

이것은 `ㆍ`가 단순한 점이 아니라, 기준축이 놓이기 전의 자리라는 뜻이다.

```text
dot
=
axis-before-place
=
축이 놓이기 전의 기준 자리
```

---

## 4. 모음 구조와 방향상태

모음은 고정된 부호가 아니라,  
dot과 기준축의 배치가 만드는 방향상태로 읽을 수 있다.

```text
ㅗ / ㅜ / ㅓ / ㅏ
=
고정 field 안에서
기준축과 dot의 관계가 달라지며 드러나는 방향상태
```

여기서 `ㅣ`는 사람 / 기준축이고, `ㅡ`는 땅 / 공간평면이다.

---

## 5. 구조원리와의 연결

훈민정음의 기본 요소는 구조원리의 다음 흐름과 연결될 수 있다.

```text
dot
→ axis
→ plane
→ closure
→ cycle
```

또한 다음 구조와도 관계를 가진다.

```text
difference
→ flow
→ rotation
→ circulation
→ closure
```

이 연결은 훈민정음을 문자학적으로 확정하려는 목적이 아니라,  
훈민정음 구조 안에서 구조원리의 공통 패턴을 읽기 위한 source relation이다.
