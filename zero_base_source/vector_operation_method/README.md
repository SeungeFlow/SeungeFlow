# vector_operation_method

`vector_operation_method`는 한국어로 `벡터연산기법`으로 읽는다.

이 분류는 훈민정음 해례본 제자원리를 구조원리로 내린 흐름을 보존하기 위한 자리이다.

```text
벡터연산기법
=
훈민정음 해례본 제자원리
→ 구조원리의 dot / axis / plane / closure / flow 구조
```

---

## 핵심 reading

```text
ㆍ = 기준이 놓일 자리 / dot
ㅣ = 기준축 / vertical axis
ㅡ = 공간평면 / horizontal axis
ㅇ = 닫힘 / closure / boundary
```

이 요소들은 단순 문자요소가 아니라, 구조가 드러나는 최소 구성요소로 읽는다.

---
