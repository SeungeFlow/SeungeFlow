# zero_base_source

`zero_base_source`는 구조원리 이전 또는 구조원리의 바닥을 이루는 Source 묶음을 모아 읽기 위한 분류이다.

이 디렉토리는 특정 응용 결과물이 아니라, 현시점 구조원리의 Source 격 문서묶음을 보존하고 관계를 정리하기 위한 자리이다.

```text
zero_base_source
=
Ctp_당연한이론
+
벡터연산기법
+
sejong_hangul
```

---

## 1. Ctp_ObviousTheory

`Ctp_ObviousTheory`는 한국어로 `Ctp_당연한이론`으로 읽는다.

```text
Ctp_당연한이론
=
구조원리 이전의 Zero.Base 상태로 회귀하여
바닥을 다시 다지는 이론
```

이 문서묶음은 현시점 구조원리의 Source 격으로 둔다.

---

## 2. vector_operation_method

`vector_operation_method`는 한국어로 `벡터연산기법`으로 읽는다.

```text
벡터연산기법
=
훈민정음 해례본 제자원리를
구조원리로 내린 기법
```

이 흐름에서는 훈민정음의 `ㆍ`, `ㅣ`, `ㅡ`, `ㅇ`을 단순 문자요소로만 보지 않고, 구조원리의 dot / axis / plane / closure / flow 구조와 연결해 읽는다.

---

## 3. sejong_hangul

`sejong_hangul`은 훈민정음과 구조원리의 관계를 따로 모아 읽는 하위 분류이다.

```text
sejong_hangul
=
훈민정음 관련 source field
+
세종 / 훈민정음 / 천지인 / 제자원리 / 구조원리 mapping
```

---

## 4. 하위 구조

```text
zero_base_source/
├── Ctp_ObviousTheory/
├── vector_operation_method/
└── sejong_hangul/
```

---

## 5. 주의할 점

이 디렉토리의 문서들은 곧바로 완성된 `meta.md`로 읽지 않는다.

이곳의 Seed들이 기존 개념에 덧붙는 형태인지, 새 meta.md로 추가되는 형태인지는 이후 별도 판단한다.

Source boundary를 보존한 상태로 relation을 확인한다.
