## 3. 사각 시작과 내부 마름모 드러남

이번 spreadsheet rendering에서 기존 수열과 다른 점은 첫 시점의 패턴이 사각으로 시작했다는 것이다.

```text
start pattern = square
revealed inner pattern = rhombus
```

4개의 정사각 전부를 같은 숫자패턴으로 배치하고, 각 정사각의 바깥경계선을 같은 숫자 흐름으로 맞추면 내부에는 마름모 경계를 가진 상태가 드러난다.

반대로, 첫 시작 패턴을 마름모로 설정했다면 내부에 드러나는 것은 사각패턴일 수 있다.

```text
start pattern = rhombus
revealed inner pattern = square
```

사각형 두 개가 겹치는 것이 아니라, 끝점이 대각으로 대응되는 상태이다.
