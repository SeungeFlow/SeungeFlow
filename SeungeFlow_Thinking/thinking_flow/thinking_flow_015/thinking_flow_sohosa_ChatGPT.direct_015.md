# 4. 완창대군파 branch origin

완창대군파의 핵심 branch origin은 다음이다.

```text
완창대군
=
휘 자흥
+
도조대왕 제1남
+
완창대군파 계별지조
+
branch origin coordinate
```

완창대군은 후사가 없었고, 환조대왕의 장자인 천계가 완창대군에게 출계하여 승사하였다.

```text
완창대군 후사 없음
→ 환조 장자 천계 승사
→ 영성대군
→ 완창대군파 2世
```

여기서 핵심은 다음이다.

```text
후사 없음
=
biological direct-line interruption

승사
=
ritual-lineage continuity restoration

계자 / 출계
=
branch identity preservation operation
```

따라서 완창대군파는 단순 생물학적 직계로만 읽으면 안 된다.  
완창대군파는 `계별지조 / 승사 / 계자 / 향사 / 부조묘 / 소호사 / 세덕사`가 함께 작동하는 ritual-lineage continuity field이다.
