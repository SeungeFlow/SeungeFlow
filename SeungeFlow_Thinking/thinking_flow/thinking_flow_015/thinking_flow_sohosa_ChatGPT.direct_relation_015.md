# 1. 생성 이유

`thinking_flow_sohosa_ChatGPT.direct_015.md`는 전주이씨 완창대군파 / 소호사 / 안양군 세파 / 항렬 / 이름-field / 관계 / 계 / 천체물리학 relation을 통해 많은 Seed를 포함하고 있다.

하지만 그 Seed들을 곧바로 `meta.md`로 올리면 collapse 위험이 있다.

```text
collapse 위험 =
candidate를 confirmed로 읽음
+
relation을 merge로 읽음
+
source layer를 schema layer로 흡수함
+
pending을 임의로 닫음
+
reference_only를 proof claim으로 오독함
```

따라서 다음 단계가 필요하다.

```text
thinking_flow
→ relation 문서
→ meta.md 후보 판정
→ meta.md 생성 / 기존 meta.md 연결 / pending 보존
```

이 문서가 생성되어야 다음을 이을 수 있다.
