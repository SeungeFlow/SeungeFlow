## 1. C+1의 현재 위치

현재 Einstein-Relativity 세트의 중심식은 다음이다.

```text
Context_Einstein_C + Event_Relativity_C = C+1
```

여기서 `+`는 단순 덧셈이 아니다.

```text
+ =
relation
```

따라서 C+1은 Context와 Event를 합쳐 만든 결론문이 아니다.

C+1은 다음이다.

```text
C+1 =
Context_C와 Event_C 사이의 relation path가
새로운 구조 후보를 여는 상태
```

현재 판정:

```text
C+1 =
provisional candidate
final judgment 아님
```
