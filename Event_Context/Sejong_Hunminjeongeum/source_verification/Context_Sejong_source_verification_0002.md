## 0. 이 문서의 자리

이 문서는 `Context_Sejong.md`의 source verification 2회차 문서다.

1회차 source verification은 주로 `Event_Hunminjeongeum`의 창제 표식, 완성 / 문헌 표식, 해례본 구성 source anchor를 검산했다.

이번 2회차는 `Context_Sejong` 쪽 source를 검산한다.

```text
검산 대상 =
세종 생애
왕위 형성
조선 초기 국가운영장
집현전 / 학문장
```

이 문서는 최종 역사해석문이 아니다.

이 문서는 source anchor들이 `Context_Sejong`의 m / t / p / ? 중 어디를 지지하는지 확인하는 문서다.

```text
source
→ m / t / p / ?
→ Context_Sejong_C 후보
```

## 2. Verified Source Anchor C-SJ-01 — 세종 기본 인물 정보

source:

```text
한국민족문화대백과 「세종」 항목
```

확인된 source fact:

```text
1. 세종은 조선의 제4대 왕이다.
2. 재위 기간은 1418년부터 1450년까지이다.
3. 출생 연도는 1397년이며, 출생지는 한양으로 제시된다.
4. 이름은 이도(李祹), 자는 원정(元正)으로 제시된다.
5. 태종의 셋째 아들이고, 어머니는 원경왕후 민씨로 제시된다.
6. 1418년 6월 왕세자에 책봉되었다가 같은 해 8월 태종의 양위를 받아 즉위한 것으로 설명된다.
```

Ctp24 mapping:

```text
maps_to =
Context_Sejong_m
+
Context_Sejong_t
```

검산 판정:

```text
C-SJ-01은
Context_Sejong의 중심 m과 왕위 형성 t를 지지하는 B급 source anchor로 적절하다.
```

주의:

```text
이 source는 세종의 기본 인물 정보와 즉위 흐름을 지지한다.
이것만으로 Context_Sejong_C 전체를 확정하지 않는다.
```
