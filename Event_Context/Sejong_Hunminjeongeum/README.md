# Sejong-Hunminjeongeum Event / Context Set

> 상태: Ctp24 Event / Context 첫 적용 초안 세트  
> 대상: Sejong / Hunminjeongeum  
> 중심식: `Context_Sejong_C + Event_Hunminjeongeum_C = C+1`

---

## 0. 이 패키지의 자리

이 패키지는 Ctp24 구조원리와 구조연산기를 실제 Event / Context 구조로 처음 작동시킨 초안 세트다.

```text
Context_Sejong.md
+
Event_Hunminjeongeum.md
+
Path_Sejong_Hunminjeongeum.md
=
첫 Event / Context operation set
```

이 패키지는 최종 역사해석이 아니다.

이 패키지는 source-aware structural draft다.

---

## 1. documents/

```text
documents/Context_Sejong.md
=
Sejong을 존재 형성장 Context_C로 읽는 문서

documents/Event_Hunminjeongeum.md
=
Hunminjeongeum을 구조 형성물 Event_C로 읽는 문서

documents/Path_Sejong_Hunminjeongeum.md
=
Context_Sejong_C와 Event_Hunminjeongeum_C 사이의 relation path를 여는 문서
```

---

## 2. schema/

schema 폴더에는 Event / Context 적용을 가능하게 만든 준비문서와 operation rule이 들어 있다.

---

## 3. guard

```text
숫자패턴은 relation 증거가 아니다.
Context와 Event를 병합하지 않는다.
C+1은 아직 최종 확정하지 않는다.
source와 구조해석을 구분한다.
```
