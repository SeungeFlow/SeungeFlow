> 대상 branch: `epluone`  
> 대상 위치: `Event_Context/Sejong_Hunminjeongeum/`

---

## 0. 이 패키지의 자리

이 패키지는 Sejong-Hunminjeongeum Event / Context 첫 적용 세트를 새로 만드는 패키지가 아니다.

이 패키지는 이미 epluone branch에 반영된 다음 위치를 업데이트하기 위한 패키지다.

```text
epluone/Event_Context/Sejong_Hunminjeongeum/
```

업데이트 핵심은 다음이다.

```text
1. source verification 문서 추가
2. Path_Sejong_Hunminjeongeum.md를 source verification 결과가 반영된 update draft로 교체
3. operation judgment 문서 추가
```
