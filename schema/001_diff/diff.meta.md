# diff.meta.md

## Definition

```text
diff = 관측기준 ?에 의해 m의 실체가 바뀌는 차이구조
```

## Same-axis case

```text
dotㅡdotㅡdot
```

```text
dot 3개
선분 2개
1차원 연속구조
m = dot
```

## Axis-transition case

```text
dotㅡdotㅣdot
```

```text
dot 3개
선분 3개
분기구조
m = dotㅡ
```

`?`가 바뀌면 `m`도 바뀐다.

```text
? = 동일축 기준 → m = dot
? = 축변환 기준 → m = dotㅡ
```
