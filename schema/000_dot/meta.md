# dot.meta.md

## Definition

```text
dot = 현재 관측기준에서 하나의 점으로 압축되어 보이는 구조상태
```

`dot`은 고정된 점이 아니다.

```text
dot = point / circle / n-gon / line-state / plane-state / galaxy / cluster / compressed field
```

## Observer Criterion

```text
same visual dot + different observation condition → different m
```

관측거리, 해상도, 축, field boundary, 관측기준 `?`가 바뀌면 dot의 실체가 바뀐다.

## dot dot

```text
ㆍ ㆍ = 서로 다른 m값, 축, place-state를 가진 두 dot 구조 사이의 차이
```

`ㆍ ㆍ`는 단순히 같은 점 두 개가 아니다. 차이와 임계사이영역을 포함할 수 있다.
