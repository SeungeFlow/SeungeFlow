# dimension.meta.md

## Definition

```text
차원 = 고정된 공간 수가 아니라, 관측기준 / 해상도 / 거리 / 경계 / m 설정에 따라 formed 되는 해석상태
```

## Layer interpretation

```text
1단 = 점의 연결일 수도 있고, 선분의 연결일 수도 있고, 면이나 공간의 연결일 수도 있다.
10단 = 현재 해석기준에서 m을 10개 단위로 배열한 구조수열.
```

가까운 관측에서는 10단 구조수열이 보이고, 상위 관측에서는 그 전체가 하나의 dot이 될 수 있다.

## Guard

구조수열은 source fact가 아니라 structure interpretation일 수 있다.
