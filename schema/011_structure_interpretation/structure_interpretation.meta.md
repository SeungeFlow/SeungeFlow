# structure_interpretation.meta.md

## Definition

```text
구조해석 = 어떤 구조에 의미를 부여하는 작업
```

C=(m,t,p,?)는 m, t, p, ? 각각에 관측기준에 따른 의미를 부여하는 구조해석식이다.

## Source fact vs interpretation

```text
source fact ≠ structure interpretation
```

예:

```text
10단 구조수열 = 오감도 시제4호의 비틀어진 구조를 똑바로 배치해 보며 만든 구조해석 배열
```

이는 source fact가 아니라 structure interpretation이다.
