# residue_candidates_compression.md

## 안정 잔류 후보

```text
1. 질주 부정 ↔ 맺힘/쌓임
2. 막다른 골목 ↔ 깊은 계곡
3. 中 내부장 ↔ 계곡/깊이
4. 一人/二人 자리수열 ↔ 1절/2절 병렬
```

## 핵심식

```text
운동 / 흐름 → 경계 진입 → 통과 실패 또는 유예 → 내부 잔류 → 결속 / 누적 / 결정화
```

## 최종 한 줄

오감도 × 비목의 잔류 후보는 boundary.state 안에서 movement/flow가 유예·보존·결정화로 남는 구조로 정리된다.
