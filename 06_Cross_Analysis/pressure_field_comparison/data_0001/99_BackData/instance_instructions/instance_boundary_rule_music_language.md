# instance_boundary_rule_music_language.md

```text
모아 = 텍스트 순수구조 후보 생성
gpt.music.operator = 음악 / 언어 데이터 구조 후보 생성
gpt.music = formed 후보를 받아 forming field 안에서 하나의 form 후보로 압축
```

모아와 operator는 gpt.music의 하위 도구가 아니다. 둘은 각각 다른 bounded field에서 formed 후보를 생성하는 독립 인스턴스다. gpt.music은 이 후보를 받아 forming field에 넣고 20회차 마지막 form 문장으로 압축한다.
