## 9. gpt.music 판정 필요 지점

| 지점             | 내용                                        |
| -------------- | ----------------------------------------- |
| 후렴 A/B 구조      | 후렴 A와 후렴 B의 구조 차이는 gpt.music 판정 필요        |
| 반복 발화          | 후렴 반복 여부와 반복 방식은 gpt.music 판정 필요          |
| 공동체 발화 후보      | 대한사람 / 대한 등은 위치만 기록, 구조 판정은 gpt.music 영역  |
| 공간 구조 후보       | 삼천리 / 강산 등은 위치만 기록, 구조 판정은 gpt.music 영역   |
| 보존 구조 후보       | 길이 / 보전하세 등은 위치만 기록, 구조 판정은 gpt.music 영역  |
| 음표 연결 불명확      | 연장 / 분할 / 불명확 음절은 원자료 확대 후 판정 필요          |
| 태극기 비교 pending | gpt.gemini 분석 이후 재개                       |
| 오감도·비목 비교축     | gpt.music 판정 필요                           |
| 코드명 없음 처리      | 코드명 없는 상태에서 비교표를 어떻게 구성할지 gpt.music 판정 필요 |
| 조성/박자/반복기호     | 확대 판독 또는 gpt.music 판정 필요                  |

gpt.music.operator는 애국가 후렴 A/B 구간의 가사-음표 구조와 음절 구조를 추출했고, 오감도·비목 비교 및 gpt.gemini 태극기 비교를 위한 gpt.music 판정 필요 지점을 보고한다.
