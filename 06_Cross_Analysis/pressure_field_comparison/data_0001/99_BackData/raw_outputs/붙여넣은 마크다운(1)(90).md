## 2.2 말미 병렬 구간 고정 구조

```text
1절:
마디마디 → 이끼 → 되어 → 맺혔네

2절:
알알이 → 돌이 → 되어 → 쌓였네
```

압축 구조:

```text
반복/분절
→ 잔류/고정 구조어
→ 공통 전환
→ 결속/누적
→ 전환성 종결
```

## 7. 전체 구조 흐름 압축

비목 전체 가사는 다음 흐름으로 1차 구간화된다.

```text
전주 / 진입 준비
→ 초반 공간 구간
→ 중반 시간·대상 구간
→ 후반 기억·거리 구간
→ 말미 병렬 결속·누적 구간
```

1절 흐름:

```text
공간/경로
→ 시간/이름/대상
→ 거리/하늘/기억
→ 마디마디/이끼/되어/맺힘
```

2절 흐름:

```text
산/빛/밤
→ 바람/상태/대상
→ 옛날/추억/달
→ 알알이/돌/되어/쌓임
```
