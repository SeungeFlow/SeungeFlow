## 6. 1차 테스트 구간 후보

| 후보   | 가사 구간           | 선택 이유                                                            | gpt.music 판정 필요 |
| ---- | --------------- | ---------------------------------------------------------------- | --------------- |
| 후보 A | 1절 말미의 이끼/맺힘 구간 | 비목 구조 후보 중 `이끼`, `맺힘`이 함께 나타남. 오감도와 비교할 때 정지·잔류·쌓임 계열 후보로 둘 수 있음 | 예               |
| 후보 B | 2절 말미의 돌/쌓임 구간  | 1절 말미와 같은 악보 위치에 병렬로 배치된 구조 후보. `돌`, `쌓임` 축 확인 가능                | 예               |
| 후보 C | 초반의 깊은 계곡 구간    | 비목 구조 후보 중 `계곡`이 가장 먼저 나타나는 위치. 오감도의 도로/골목 구조와 경로 비교 가능          | 예               |

### operator 임시 추천

| 우선순위 | 후보   | 이유                                                       |
| ---: | ---- | -------------------------------------------------------- |
|    1 | 후보 A | 1절 기준으로 먼저 처리하기 적합함. 이끼/맺힘 구조가 포함되어 비목의 압력장 구조 후보를 잡기 좋음 |
|    2 | 후보 B | 후보 A와 병렬 위치에 있어 2절 비교 확장에 적합함                            |
|    3 | 후보 C | 오감도의 도로/골목과 비교하기 좋으나, 비목 내부의 맺힘/쌓임 구조를 보기에는 후보 A/B보다 약함  |

이번 회차에서는 후보만 설정한다.
음표-가사 정렬표는 다음 회차에서 수행한다.

## 7. 오감도 비교 가능 구조축

| 비목 구조 후보 | 오감도 대응 후보        | 메모                     |
| -------- | ---------------- | ---------------------- |
| 계곡       | 골목               | 닫힌/깊은 경로 구조 비교 후보      |
| 산        | 도로/경로            | 공간 배경 또는 진행 경로 후보      |
| 세월       | 숫자 진행 / 반복       | 시간 누적 구조 비교 후보         |
| 이름       | 반복 발화 단위         | 반복되는 언어 단위 후보          |
| 비목       | 아해 반복 단위         | 반복 중심어 후보              |
| 이끼       | 잔류 / 부착 / 표면 누적  | 오감도의 질주 구조와 반대되는 잔류 후보 |
| 돌        | 고정 / 응결 / 물질화 후보 | 질주/운동 구조와 대비 가능        |
| 맺힘       | 질주 부정 / 닫힘       | 운동의 정지·결속 후보           |
| 쌓임       | 반복 누적            | 번호 반복과 누적 구조 비교 후보     |

비교는 아직 수행하지 않는다.
이번 회차에서는 비교 가능한 구조축의 위치만 표시한다.
