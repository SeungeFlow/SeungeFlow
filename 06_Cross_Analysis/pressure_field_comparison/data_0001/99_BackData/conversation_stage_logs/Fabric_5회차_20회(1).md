# 6. 비목 결정화 구조 핵심식

비목의 전체 흐름은 다음이다.

```text
공간 / 경로 / 깊이
→ 시간 / 대상 / 상태
→ 기억 / 거리 / 회귀
→ 반복 / 잔류 / 고정 / 결속 / 누적
```

이를 말미 중심으로 압축하면 다음이다.

```text
공간이 먼저 열린다.
시간과 대상이 놓인다.
기억과 거리가 형성된다.
반복 단위가 생긴다.
흐름이 남고 붙고 굳는다.
마지막에 맺히고 쌓인다.
```

구조식:

```text
비목
= 공간·시간·기억의 층을 지나
  말미에서 결속과 누적으로 결정화되는 반복 구조
```
