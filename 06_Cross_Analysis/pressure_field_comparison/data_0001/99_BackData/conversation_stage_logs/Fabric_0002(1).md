# gpt.music 20회차 1차 마무리 계획

## 0. 목표

이 계획은 전체 SeungeFlow를 완성하려는 계획이 아니다.

목표는 다음이다.

```text
현재 gpt.music이 진행 중인 music_language / 오감도 × 비목 / 구조겹침 실험을
20회차 안에서 1차 마무리한다.
```

1차 마무리란 다음을 뜻한다.

```text
모든 분석을 끝내는 것 아님
모든 곡과 모든 구조를 처리하는 것 아님
특이점 / 공명 / 정신세계 잔류 최종 판정 아님

현재까지 열린 구조를
하나의 place.state에 놓을 수 있는 수준으로 정리하는 것
```
