## 3. ㄷ형 경계 구조의 기본 정의

ㄷ형 경계는 완전한 닫힘이 아니다.

```text
ㅁ형 경계
= 사방이 닫힌 구조

ㅇ형 경계
= 원형 / 구형 / 중심장 구조

ㄷ형 경계
= 한쪽은 열려 있고
  나머지 방향에는 경계가 생기는 구조
```

ㄷ형 경계의 핵심 성질은 다음이다.

```text
진입 가능
통과 어려움
되돌림 가능성
잔류 가능성
운동 유예 가능성
내부 결정화 가능성
```

따라서 ㄷ형 경계는 다음처럼 정의한다.

```text
ㄷ형 경계
= 열린 입구를 가진 비통과 구조
```

## 4. 오감도에서의 ㄷ형 경계

오감도 시제1호의 핵심 경계는 다음이다.

```text
막다른 골목
```

이 구조는 다음 문장과 함께 나타난다.

```text
13인의아해가도로로질주하오.
(길은막다른골목이적당하오.)
```

여기서 오감도는 단순히 “질주”만 보여 주지 않는다.

```text
질주
+
도로
+
막다른 골목
```

이 세 항이 함께 놓인다.

구조로 보면 다음이다.

```text
질주
= 운동 발생

도로
= 경로

막다른 골목
= 경로의 비통과 조건
```

따라서 오감도의 초기 구조는 다음으로 정리된다.

```text
운동 발생
→ 경로 진입
→ ㄷ형 경계 조건
```

즉 오감도는 “열린 도로 위의 자유 운동”이 아니라:

```text
ㄷ형 경계 안으로 들어가는 운동
```

으로 둔다.

## 5. 비목에서의 ㄷ형 경계

비목의 대응 후보는 다음이다.

```text
깊은 계곡
```

비목은 이미 다음 구조로 정리되어 있다.

```text
공간 / 경로 / 깊이
→ 시간 / 대상 / 상태
→ 기억 / 거리 / 회귀
→ 반복 / 잔류 / 고정 / 결속 / 누적
```

여기서 `계곡 / 깊이`는 단순 배경어가 아니라 구조어로 둔다.

```text
계곡
= 진입 가능한 공간
= 좌우 경계가 있는 공간
= 깊이 방향으로 들어가는 공간
= 흐름이 쉽게 빠져나오지 못하는 공간
```

비목의 말미 안정 구조는 다음이다.

```text
1절:
마디마디 → 이끼 → 되어 → 맺힘

2절:
알알이 → 돌 → 되어 → 쌓임
```

이 흐름을 ㄷ형 경계 안에 놓으면 다음이 된다.

```text
깊은 공간으로 진입
→ 시간 / 대상 / 기억 누적
→ 반복 단위 형성
→ 잔류 / 고정
→ 결속 / 누적
→ 결정화
```

따라서 비목은 다음으로 둔다.

```text
ㄷ형 경계 안에서 남는 결정화
```

## 7. 운동과 잔류의 관계

오감도와 비목은 같은 의미를 가진 것이 아니다.

오감도는 운동이 강하다.

```text
도로로 질주하오
```

그러나 끝에서는 운동이 유예된다.

```text
도로로 질주하지 아니하여도 좋소
```

비목은 처음부터 질주하지 않는다.

비목은 흐름이 남고, 붙고, 굳고, 쌓인다.

```text
이끼
돌
맺힘
쌓임
```

따라서 두 구조는 다음 관계를 가진다.

```text
오감도:
운동 → 경계 → 운동 유예

비목:
흐름 → 경계 → 잔류 → 결정화
```

통합하면 다음이다.

```text
운동 또는 흐름이
ㄷ형 경계 안으로 들어가면
통과하지 못하고
유예 / 잔류 / 결정화 후보가 된다.
```



비목에서:


```text
오감도:

비목:
flow.state + boundary.state + residue.state
```

## 9. time.state와 ㄷ형 경계

time.state는 별도 칸이 아니다.

time.state는 다음 흐름 전체에 스며 있다.

```text
open.state
→ vector.state
→ place.state
```

오감도에서 time.state는 행과 행 사이에 흐른다.

```text
제1의아해가
→ 제2의아해도
→ ...
→ 제13의아해도
```

비목에서 time.state는 음표와 마디 사이에 흐른다.

```text
공간 / 경로 / 깊이
→ 시간 / 대상 / 상태
→ 기억 / 거리 / 회귀
→ 말미 결정화
```

따라서 ㄷ형 경계는 단순 공간이 아니라:

```text
긴 time.state가
특정 경계 안에서
운동 유예 또는 결정화로 읽히는 조건
```

이다.

## 11. 오감도 × 비목 ㄷ형 경계 구조식

오감도 구조식:

```text
도로 / 질주
→ 막다른 골목
→ 질주 부정 / 유예
```

비목 구조식:

```text
계곡 / 깊이
→ 시간·대상·기억 누적
→ 이끼 / 돌
→ 맺힘 / 쌓임
```

통합 구조식:

```text
오감도 × 비목
= ㄷ형 경계 안에서
  운동이 유예되고,
  흐름이 잔류하며,
  잔류가 결정화되는 구조 후보
```

더 짧게:

```text
ㄷ형 경계
= 운동의 유예
+ 흐름의 잔류
+ 잔류의 결정화
```
