## 5.2 가/도 반복 구조

오감도 제1~제13 반복에서 핵심은 다음이다.

```text
제1의아해가
제2~제10의아해도

제11의아해가
제12~제13의아해도
```

구조식:

```text
가 → 도×9
+
가 → 도×2
```

또는:

```text
1 + 9
+
1 + 2
```

판정:

```text
가 = open.state / 기점 설정
도 = continuation.state / 중첩 추가
```

안정 후보로 둔다.

## 5.3 조사 구조의 현재 판정

```text
오감도의 반복은 단순 숫자 반복이 아니라,
가/도 조사 전환을 통해
```

로 둔다.

## 6.2 一人 / 二人 구조

핵심 수열:

```text
一人
二人
二人
一人
```

이것은 아라비아 숫자 `1 / 2 / 2 / 1`로만 환원하지 않는다.

층 분리:

```text
수량층:
1 → 2 → 2 → 1

한자 형태층:
一人 → 二人 → 二人 → 一人

상태층:
무서운 → 무서운 → 무서워하는 → 무서워하는

자리층:
P1 → P2 → P3 → P4
```

현재 판정:

```text
一人/二人/二人/一人은 숫자값이 아니라
中 내부장 안에 놓인 자리수열 구조다.
```

안정 후보로 둔다.

# 7. 4끝점 구조 후보

## 7.1 A-B-D-C 배치

현재 가장 안정적인 4끝점 배치는 다음이다.

| 자리 | 원문 구조         | 구조 역할       |
| -- | ------------- | ----------- |
| A  | 一人 / 무서운 兒孩   | 외곽 1 / 상태 1 |
| B  | 二人 / 무서운 兒孩   | 내부 2 / 상태 1 |
| D  | 二人 / 무서워하는 兒孩 | 내부 2 / 상태 2 |
| C  | 一人 / 무서워하는 兒孩 | 외곽 1 / 상태 2 |

텍스트 순서:

```text
A → B → D → C
```

구조 해석:

```text
A → B
= 수량 증가

B → D
= 상태 전환

D → C
= 수량 감소
```

판정:

```text
A-B-D-C는 오감도 내부 부분집합의 텍스트 순서 후보로 유지한다.
```

---

## 7.2 B-A-DㆍC 관계 구조

관계 구조에서는 다음 후보를 둔다.

```text
B → A → D ㆍ C
```

의미:

```text
B-D 중심축을 먼저 보고,
A-C 외곽축은 그 중심 전환을 통해 읽는다.
```

현재 판정:

```text
B-A-DㆍC는 관계 재배열 후보로 유지한다.
```

단, 최종 확정은 보류한다.
