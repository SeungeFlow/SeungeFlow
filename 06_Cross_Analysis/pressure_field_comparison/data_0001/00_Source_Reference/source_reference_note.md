# source_reference_note.md

## 오감도

1차 기준은 사용자가 대화창에 제공한 오감도 전문이다. 위키문헌, 위키백과, 한국민족문화대백과는 보조 검산 자료로 사용한다.

## 비목

비목 악보는 사용자가 정당 구매한 분석 원자료다. 공개 GitHub에는 원본 PDF, 이미지, 악보 사이트 원본 파일을 포함하지 않는다.

## 분석 목적

오감도와 비목이 의미가 같다는 주장을 하는 것이 아니라, field 안에서 repetition, boundary, sequence, residue 구조가 어떻게 공유·충돌·미끄러짐·잔류하는지 본다.
