# d_boundary_structure_ogamdo_bimok.md

## 정의

```text
ㄷ형 경계 = 열린 입구를 가진 비통과 구조
```

## 오감도

```text
도로 / 질주 → 막다른 골목 → 질주 부정 / 유예
```

## 비목

```text
계곡 / 깊이 → 시간·대상·기억 누적 → 이끼 / 돌 → 맺힘 / 쌓임
```

## 통합 구조식

```text
오감도 × 비목
= ㄷ형 경계 안에서 운동이 유예되고, 흐름이 잔류하며, 잔류가 결정화되는 구조 후보
```

## 최종 한 줄

오감도와 비목의 ㄷ형 경계 구조는 막다른 골목과 깊은 계곡을 통해 가장 강하게 겹치며, 오감도는 ㄷ 안으로 들어가는 운동, 비목은 ㄷ 안에서 남는 결정화로 정리된다.
