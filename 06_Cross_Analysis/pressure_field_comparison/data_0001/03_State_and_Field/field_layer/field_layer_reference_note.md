# field_layer_reference_note.md

## 대상간 차원

차원은 시간적 차원만이 아니라 field 안에서 대상과 대상이 포함·반전·겹침·분리되는 관계층위다.

## 비목 field

```text
비목 전체 field
├─ 1절 field
├─ 2절 field
└─ 단어 / 구절 / 음율 field
```

## 오감도 field

```text
오감도 전체 field
├─ 13인의 아해 field
├─ 그中에 field
├─ 괄호 field
└─ 외곽 운동 field
```

## music-language field

```text
가사 field + 음계/음가/마디/선율 field = music-language field
```
