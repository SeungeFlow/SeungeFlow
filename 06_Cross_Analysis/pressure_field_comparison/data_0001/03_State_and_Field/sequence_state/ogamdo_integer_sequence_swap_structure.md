# ogamdo_integer_sequence_swap_structure.md

## 대상

```text
그中에一人의兒孩가무서운兒孩라도좃소.
그中에二人의兒孩가무서운兒孩라도좃소.
그中에二人의兒孩가무서워하는兒孩라도좃소.
그中에一人의兒孩가무서워하는兒孩라도좃소.
```

## 층 분리

```text
수량층: 一人 → 二人 → 二人 → 一人 = [1,2,2,1]
상태층: 무서운 → 무서운 → 무서워하는 → 무서워하는 = [0,0,1,1]
자리층: A → B → D → C
```

## 판정

```text
수량층은 보존된다: R q = q
상태층은 반전된다: R s = [1,1,0,0]
```

## 최종 한 줄

오감도 시제1호의 一人→二人→二人→一人은 숫자값이 아니라 中 내부장 안에 놓인 자리수열이며, 수량층은 보존되고 상태층은 반전되는 sequence-swap 구조로 정리된다.
