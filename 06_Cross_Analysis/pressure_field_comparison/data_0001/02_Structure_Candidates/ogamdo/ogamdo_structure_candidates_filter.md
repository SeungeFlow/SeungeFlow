# ogamdo_structure_candidates_filter.md

## 안정 후보

```text
가 = 관측 대상의 기점 설정
도 = 기존 관측 구조에 중첩 추가
中 = 내부장 / 중심장
一人→二人→二人→一人 = 자리수열
A-B-D-C = 텍스트 순서 후보
수량층 1→2→2→1 = 대칭 보존 후보
상태층 무서운→무서워하는 = 반전 대응 후보
막다른 골목 = ㄷ형 경계
질주하지 아니하여도 좋소 = 운동 유예 / 부정
```
