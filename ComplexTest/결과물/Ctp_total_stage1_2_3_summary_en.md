# 1. Final Position of Ctp

Ctp is a function.

```text
C = Ctp(t, P_place, m, ?)
```

But Ctp is not a function that directly proves mathematical problems.

```text
Ctp = structural-reading function
```

Ctp opens the words and structures of a problem through language, compresses them into mathematics, checks them through code, and then verifies whether the three layers correspond to the same structure principle in `Structure_Principle/schema`.

# 7. Three-Layer Method

Every problem is handled through the following procedure:

```text
1. Extract key English terms.
2. Convert them into Korean structural words.
3. If needed, analyze etymology, origin, Hanja, and Hunminjeongeum letter principles.
4. Run the reverse audit: ?→m→P_place→t→C.
5. Write the language layer.
6. Compress into the mathematical layer.
7. Write a structural coding layer.
8. Map to Structure_Principle/schema.
9. Judge as pass, correction, pending, or not applicable.
```

The layers must match:

```text
language layer = mathematical layer = coding layer
```
