# R12_field_question_validation.md

## Definition

```text
R12 = Field-Question Validation
```

R12는 구조연산식으로 넘어가기 전 source identity, 관측기준, boundary, final judgment 위험, overinterpretation 위험을 검증한다.

## Role

R12가 없으면 cross는 의미 등치, 과잉해석, source 병합, final judgment 오선언으로 미끄러질 수 있다.

## Checks

```text
source identity가 고정되었는가?
source fact와 structure interpretation을 분리했는가?
field 병합이 발생했는가?
provisional candidate를 final judgment로 올렸는가?
noise를 실체로 오인했는가?
```
