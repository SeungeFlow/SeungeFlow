# 4corner_pin_README

## 1. 개념

`4corner_pin.zip` 내부 자료는 다음 구조군을 하나의 공통 구조로 다룬다.

- 태극 및 건·곤·감·리 구조
- 4-Corner Pin + Center 구조
- 음양오행 배치
- 훈민정음 모음 구조
- 대류 및 자연 순환 구조

이 자료군의 중심 개념은 서로 다른 명칭의 대상들을 비유적으로 느슨하게 연결하는 것이 아니라, 하나의 공통 구조 위에서 다시 읽는 데 있다.

---

## 2. 정의

이 자료 내부에서 반복적으로 정의되는 기준은 다음과 같다.

### 2.1 태극 · 건곤감리

- 긴 선 = 양
- 짧은 선 2개 = 음
- 건 = 양·양·양
- 곤 = 양·음·양
- 감 = 음·양·음
- 리 = 음·음·음

이 배열은 단순 방위 이름이 아니라 상태 전이 순서로 읽힌다.

### 2.2 4-Corner Pin

- A.pin = 좌상 = 물
- B.pin = 좌하 = 불
- C.pin = 우상 = 금속
- D.pin = 우하 = 흙
- Center = 나무

### 2.3 Center 입체화

Center를 수직 방향으로 들어 올렸을 때 생성되는 기본 삼각 구조:

1. XAB
2. XBD
3. XAC
4. XCD

즉 평면 배치를 입체 구조로 전환하는 최소 단위로 정의된다.

### 2.4 훈민정음 모음 구조

- ㆍ = 중심점 / 발생점 / 매개점
- ㅡ = 수평 기준축
- ㅣ = 수직 기준축
- ㅗ = 위를 향하지만 안으로 머무는 소리 상태
- ㅜ = 아래를 향하지만 밖으로 나가는 소리 상태

## 3. 원리

이 자료가 사용하는 내부 원리는 다음과 같이 정리된다.

### 3.1 중심과 사방의 관계

Center는 A, B, C, D와 각각 연결되며, 중심을 들어 올릴 때 평면이 입체화된다.
이는 점-면-입체 전환 원리다.

### 3.2 대립 구조와 투영

정면 투영에서 XBD와 XAC는 서로 대립하는 삼각 구조로 읽히며,
전체는 중앙선을 가진 마름모 구조로 파악된다.

### 3.3 상승/하강 순환

- 붉은 영역 = 상승
- 푸른 영역 = 하강

태극은 상승과 하강의 순환을 나타내는 추상화 도형으로 읽힌다.
즉 건곤감리는 고정 상징이 아니라 상태 변화 흐름으로 해석된다.

### 3.4 소리와 공간의 구조 대응

모음 ㆍ, ㅡ, ㅣ, ㅗ, ㅜ는 단순 문자 기호가 아니라,
중심-축-방향-진입-이탈의 구조적 기능을 가진 것으로 놓인다.

## 4. 본질

이 자료의 본질은 다음 문장으로 정리된다.

문자, 소리, 공간, 자연현상은 서로 다른 대상이지만,
이 자료 안에서는 하나의 공통 구조 언어를 공유하는 것으로 다뤄진다.

즉 이 자료는:

- 유사 개념 수집이 아니라
- 구조 정렬 문서이며
- 태극 / 건곤감리 / 4-Corner Pin / 훈민정음 모음 / 대류 순환을
  하나의 구조계 안에서 결속시키려는 시도다.

핵심은 “비슷하다”가 아니라 “동형 구조로 읽힌다”는 데 있다.

## 5. 현상태

현재 압축 내부에서 확인된 자료는 다음과 같다.

- 설명 텍스트 1개
- 구조 정합 문서 1개 (`taegeuk_4corner_hunmin_structure_v1-1.md`)
- 이미지 파일 2개

현재 상태는 다음으로 볼 수 있다.

- 기본 개념은 이미 내부적으로 정리되어 있음
- 중심 문서는 `taegeuk_4corner_hunmin_structure_v1-1.md`
- 텍스트 파일은 발화형 원문에 가까움
- 이미지는 구조 보조 자료로 보임
- 즉, 개념 초안 + 정합 문서 + 시각 자료가 함께 있는 상태

다만 아직 이 자료군 전체를 위한 상위 README는 별도로 없었기 때문에,
현재 파일 `4corner_pin_README.md`는 그 상위 진입 문서 역할을 수행한다.

## 6. 활용가치

이 자료군의 외부 활용 가능성은 다음과 같다.

### 6.1 구조해석기 입력 자료

태극 / 건곤감리 / 훈민정음 / 4-Corner Pin의 관계를 구조 단위로 정의한 자료이므로,
구조해석기 또는 개념 그래프 입력 자료로 활용 가능하다.

### 6.2 시각화 확장

Center-A/B/C/D 관계와 삼각면 구조는
2D 도식, 3D 모델, 방향성 다이어그램으로 확장 가능하다.

### 6.3 문서 연결 허브

이 자료는 다음 종류의 문서와 연결될 때 활용도가 높다.

- 훈민정음 구조 문서
- 음양오행 배치 문서
- 태극 및 대류 구조 문서
- 3D 구조 모델링 문서
- 구조언어/기호 정의 문서

### 6.4 Navigation Map 요소

하나의 독립 문서라기보다,
향후 `MyData` 또는 상위 Navigation Map 안에서
다른 구조문서들을 연결하는 중간 허브 문서로 활용 가능하다.
