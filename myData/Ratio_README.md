# Ratio자료_README

## 1. 개념·정의

`Ratio자료`는 서로 다른 시간축과 기준선 사이의 간격을 비교하여  
현재 위치, 배열, 교차 가능성, 임계 상태를 읽기 위한 **비율분석 원리 자료 묶음**이다.

이 자료에서 말하는 Ratio는 단순한 숫자 비율이 아니라,  
특정 기준값을 중심으로 **간격의 크기와 그 간격이 만들어내는 배열 상태**를 읽는 방식으로 정의된다.

핵심 정의는 다음과 같다.

- 비율은 절대값 자체가 아니라 **기준축 대비 간격의 상대적 위치**이다.
- 분석 대상은 단일 값이 아니라 **20 / 60 / 120 / 240** 같은 기준선 묶음이다.
- RatioA / RatioB / RatioC 는 서로 다른 구간 차이를 읽기 위한 단계적 비율값이다.
- Ratio.MRI 는 개별 비율값을 넘어 **배열, 교차, 경우의 수, 상태 분기**를 보기 위한 확장형 구조이다.
- 이 자료는 가격 그 자체를 예측하는 문서라기보다, **현재 위치와 구조 상태를 해석하기 위한 계산 원리 묶음**이다.

내부 파일 구성은 대체로 다음 역할을 가진다.

- `비율계산기원리.xlsx`  
  : 비율계산의 가장 기초 개념과 확장 개념, MRI 개념의 원형을 담은 원리 문서
- `비율계산_Ratio(A,B,C)기본.xlsx`  
  : RatioA / B / C 기본 계산 구조와 스택형 비율분석 틀을 담은 기본 계산기
- `비율계산_확장_Ratio.MRI.xlsx`  
  : 기본 비율분석을 경우의 수, 정배열, 교차점, 설계도까지 확장한 분석기
- `Ratio.MRI_원본_비율계산기원리로분기.xlsx`  
  : 비율계산 원리에서 MRI 구조로 분기된 원본 계열 자료
- `원리분석1.png ~ 원리분석6.png`  
  : 비율분석 원리를 시각적으로 설명하는 그림 자료
- `Ratio.MRI_TradingView예시.png`, `Ratio.MRI_엑셀예시.png`  
  : 실제 화면에서 Ratio.MRI 구조가 어떻게 보이는지 보여주는 예시 자료
- `천간지지로 풀어본 비율계산기 원리.png`  
  : 비율계산 원리를 다른 상징 구조로 풀어본 해석 자료

즉 `Ratio자료`는 단일 계산기가 아니라,  
**기본 비율계산 → 확장 → MRI 구조화 → 배열/교차 해석 → 시각 예시**로 이어지는 원리 자료군이다.

## 2. 원리·본질

`Ratio자료`의 본질은 값 하나를 바로 해석하는 데 있지 않다.  
그 본질은 **기준선 간 거리와 그 거리의 배열 구조를 읽어 현재 상태를 판단하는 것**에 있다.

### 첫째, 기준선 간격 원리

이 자료의 가장 기본 원리는  
20, 60, 120, 240 같은 기준선을 두고,  
각 기준선 사이의 간격을 시간축별로 계산하는 것이다.

예를 들면:

- 20선과 60선 사이
- 20선과 120선 사이
- 20선과 240선 사이

같은 구간을 각각 다른 시간축(1시간, 4시간, 1일, 1주 등)에서 읽는다.

즉 본질적으로 Ratio는  
**선과 선 사이의 간격 구조를 수치화한 것**이다.

### 둘째, 비율값 단계화 원리

자료 안에서 반복되는 RatioA / RatioB / RatioC 는  
하나의 비율만 보는 방식이 아니라,  
구간 차이를 단계적으로 나누어 읽는 원리이다.

대표적으로:

- RatioA = 첫 구간
- RatioB = 둘째 구간
- RatioC = 셋째 구간

이런 식으로 간격을 나누고,  
그 간격이 기준값 대비 어느 정도 위치에 있는지를 본다.

즉 이 체계는 단일 평균선 비교가 아니라  
**구간별 단계 비율 구조**를 만든다.

### 셋째, 시간축 중첩 원리

이 자료는 한 시간축만 쓰지 않는다.

- 1시간
- 4시간
- 1일
- 1주

처럼 여러 축을 겹쳐 본다.

같은 20/60/120/240 구조라도  
시간축이 다르면 간격 의미가 달라지므로,  
각 시간축을 겹쳐 읽어야 현재 상태를 더 입체적으로 볼 수 있다.

즉 Ratio자료의 본질은  
**비율 계산 + 시간축 중첩 해석**이다.

### 넷째, 배열 / 교차 / 경우의 수 원리

확장 파일들과 MRI 구조에서는  
단순 비율값을 넘어 다음을 보게 한다.

- 정배열 상태인지
- 역배열 상태인지
- 교차점이 발생하는지
- 어느 선이 어느 선과 어떤 관계를 가지는지
- 현재 위치가 어떤 경우의 수에 속하는지

즉 Ratio.MRI 의 핵심은  
비율을 **상태 판독 도구**로 확장하는 데 있다.

여기서 MRI는 의료 영상처럼 내부를 직접 보는 의미가 아니라,  
겉의 가격 움직임 아래에 있는 **배열 구조를 읽는 관측 틀**로 이해할 수 있다.

### 다섯째, 분기 원리

`Ratio.MRI_원본_비율계산기원리로분기.xlsx` 라는 파일명이 보여주듯,  
이 자료는 하나의 기본 비율계산기에서 끝나지 않고  
그 원리로부터 여러 분석 방향이 분기되는 구조를 가진다.

즉 본질적으로 이 자료는:

- 기본 계산 원리
- 확장 계산 원리
- 배열 분석
- 교차 분석
- 시각화 예시

로 갈라지는 **분기형 분석 시스템**이다.

### 여섯째, 현재 위치 판독 원리

파일 내부 설명에 반복되는 문장은  
“20선의 현재위치를 알아내는 도구”라는 표현이다.

이 말은 곧 이 자료의 본질이  
미래를 단정적으로 맞히는 데 있기보다,  
**현재 위치를 구조적으로 판독하는 것**에 있다는 뜻이다.

즉 Ratio자료는 예언 도구보다  
**현재 상태 판독 도구**에 더 가깝다.

### 정리

결국 `Ratio자료`의 원리와 본질은 다음과 같이 정리할 수 있다.

```txt
비율분석의 본질
= 기준선 간격 계산
+ 구간별 Ratio 단계화
+ 시간축 중첩 해석
+ 배열 / 교차 / 경우의 수 판독
+ 현재 위치의 구조적 확인
```

즉 이 자료는 가격 데이터 위에  
**간격 구조와 배열 구조를 덧씌워 현재 상태를 읽는 분석 원리 묶음**이다.

## 3. 활용가치

`Ratio자료`의 활용가치는 단순 계산표를 넘는다.  
이 자료는 현재 상태를 구조적으로 읽고,  
서로 다른 시간축과 기준선 관계를 한 번에 파악하는 데 가치가 있다.

### 1) 현재 위치 판독 도구로 활용 가능하다.

이 자료는 “지금 어느 위치에 있는가”를 보기 좋다.

특히:

- 20선이 어디에 있는가
- 60/120/240과 어떤 간격 상태인가
- 정배열/역배열 쪽으로 기울어 있는가
- 교차 직전인지 이미 교차했는가

같은 질문에 답하기 위한 판독 도구로 쓸 수 있다.

### 2) 다중 시간축 비교 도구로 활용 가능하다.

하나의 종목이나 자산을 볼 때  
1시간 / 4시간 / 1일 / 1주를 따로 보면 흐름이 분절될 수 있다.

Ratio자료는 이 여러 시간축을 같은 기준선 체계 위에 얹어서 보게 하므로,  
**다중 시간축의 구조 비교 도구**로 활용할 수 있다.

### 3) 배열 분석과 교차 분석 도구로 활용 가능하다.

기본 Ratio 계산기를 넘어 MRI 확장형은  
배열 상태와 교차 가능성, 경우의 수를 체계적으로 보게 해준다.

즉 활용가치는:

- 정배열 확인
- 역배열 확인
- 교차 위치 탐지
- 관계 구조 분류

같은 쪽으로 확장된다.

### 4) TradingView / Excel 연결 분석에 활용 가능하다.

예시 이미지가 보여주듯,  
이 자료는 엑셀 안 계산만이 아니라 실제 차트 환경과 연결해서 볼 수 있다.

즉:

- 차트 화면에서 관측한 값
- 엑셀에서 계산한 Ratio 구조
- 그 둘을 연결한 판단

이 흐름으로 활용할 수 있다.

### 5) 비율분석 원리 교육 자료로 활용 가능하다.

이 ZIP 안에는 원리설명 이미지와 기본/확장 계산기, 원본 분기 자료가 함께 있다.

그래서 이 자료는 단순 실전용만이 아니라:

- 비율분석 원리 소개
- 기본 개념 교육
- 확장 구조 설명
- 시각 예시 제공

같은 교육용 자료로도 가치가 있다.

### 6) 구조분석 계열 시스템과 연결 가능하다.

승이가 말한 “비율분석의 핵심원리”라는 표현처럼,  
이 자료는 단순 금융 계산기가 아니라 구조분석 계열 도구로도 읽을 수 있다.

왜냐하면 여기서는  
값 자체보다 **관계 / 간격 / 배열 / 교차 / 현재 상태**를 읽기 때문이다.

즉 이 자료는 앞으로:

- 구조해석기
- 문서 기반 분석 시스템
- 상태 판독 엔진
- 조건 분기 엔진

같은 상위 시스템의 하위 모듈로도 연결될 수 있다.

## 최종 정리

```txt
Ratio자료
= 기준선 간격과 시간축 중첩을 통해
  현재 위치, 배열, 교차, 경우의 수를 판독하는
  비율분석 핵심원리 자료 묶음
```

이 자료의 핵심 가치는 다음과 같다.

- 비율을 구조로 읽게 한다
- 절대값보다 간격 관계를 보게 한다
- 시간축을 겹쳐 상태를 읽게 한다
- 배열과 교차를 판독하게 한다
- 현재 상태를 구조적으로 확인하게 한다
