## 3. 읽기 guard

```text
정답을 찾으려 하지 않는다.
relation을 찾는다.

깔끔한 요약을 만들려 하지 않는다.
자리와 흐름을 본다.

core와 application을 섞지 않는다.
support와 application을 혼동하지 않는다.
```
