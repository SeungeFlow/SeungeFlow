# AUTO_PIPELINE.md

## 목적

하나의 입력으로  
Grok → 구조 해석 → Gemini → 검증 → 최종 출력까지  
자동으로 수행하는 파이프라인

---

## 사용 방법

1. 아래 프롬프트를 그대로 복사한다
2. 분석할 문장을 넣는다
3. Grok → 결과 → Gemini로 이어서 사용한다

---

## 1단계 — Grok 실행 프롬프트

다음 규칙을 반드시 따른다:

자음 = 작용  
모음 = 상태  
단어 = 구조식  
문장 = 흐름식  

출력 형식:

1. 분해  
2. 구조식  
3. 흐름식  
4. 논리 해석  
5. 결론  

---

### 실행

다음 문장을 구조해석기로 분석하라:

[여기에 문장 입력]

---

## 2단계 — Gemini 검증 프롬프트

다음 결과를 검증하라:

- 자음이 작용으로 해석되었는가
- 모음이 상태로 해석되었는가
- 구조식이 존재하는가
- 흐름식이 존재하는가
- 의미 해석이 아닌 구조 해석인가

---

### 출력 형식

1. 오류 여부 (있음 / 없음)  
2. 오류 위치  
3. 수정된 구조식  
4. 수정된 흐름식  
5. 최종 결론  

---

## 3단계 — 자동 연결 규칙

Grok 결과 → 그대로 Gemini 입력

---

## 4단계 — 핵심 규칙

구조 > 의미  
자모 → 구조 → 흐름 → 논리  

---

## 5단계 — 한 줄

하나의 입력을 두 AI로 처리하여  
구조와 논리를 동시에 확보한다
