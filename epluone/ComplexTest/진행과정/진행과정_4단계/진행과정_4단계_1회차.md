# 4단계 1회차 - 0002 

[gpt.github 정리]

역할:
gpt.github는 GitHub 파일 업로드 작업과 GitHub view 주소 / Raw 주소 표시를 담당한다.
이 인스턴스는 이론 확장보다 파일 위치, 디렉토리 구조, 업로드 기준, Raw 주소 제공에 집중한다.


1. 관측기준자리 계열

대상 디렉토리:
Structure_Principle/schema/000_dot/
Structure_Principle/schema/100_empty_position/

Raw 주소:
https://raw.githubusercontent.com/SeungeFlow/SeungeFlow/main/Structure_Principle/schema/000_dot/
https://raw.githubusercontent.com/SeungeFlow/SeungeFlow/main/Structure_Principle/schema/100_empty_position/

성질:
이 두 디렉토리는 넓은 의미에서 관측기준이라고 불러도 무방하다.
다만 더 정확히는 관측기준 자체라기보다 관측기준을 세우기 위한 기준자리,
또는 관측기준이 성립하기 위한 자리 조건이다.

000_dot:
- 최초 기준자리
- origin seat
- 관측기준이 처음 놓일 수 있는 시작자리 조건
- 무언가가 놓일 수 있는 최소 자리

100_empty_position:
- 전환 기준자리
- available seat
- 다음 관측, 다음 배치, 다음 구조가 놓일 수 있도록 비워 둔 자리
- 없음이 아니라 다음 것을 받을 수 있는 빈자리 조건


2. 전체 흐름의 연속 계열

대상 디렉토리:
Structure_Principle/schema/101_* ~ 121_*
SeungeFlow_Thinking/thinking_flow/

Raw 주소 기준:
https://raw.githubusercontent.com/SeungeFlow/SeungeFlow/main/Structure_Principle/schema/
https://raw.githubusercontent.com/SeungeFlow/SeungeFlow/main/SeungeFlow_Thinking/thinking_flow/

성질:
Structure_Principle/schema/101부터 121까지의 디렉토리들과
SeungeFlow_Thinking/thinking_flow/ 하위 디렉토리들은
전체 구조 흐름의 연속으로 본다.

즉, 000_dot과 100_empty_position이 관측기준자리라면,
101~121과 thinking_flow는 그 기준자리 이후에 이어지는
구조 형성 흐름, 사고 흐름, 관계 흐름의 연속이다.


3. 현재 다음 관측대상

대상 디렉토리:
epluone/ComplexTest/

GitHub view 주소:
https://github.com/SeungeFlow/SeungeFlow/tree/main/epluone/ComplexTest

Raw 주소:
https://raw.githubusercontent.com/SeungeFlow/SeungeFlow/main/epluone/ComplexTest/

성질:
현재 epluone/ComplexTest 디렉토리는
Structure_Principle/schema/101~121과
SeungeFlow_Thinking/thinking_flow/의 연속 흐름 다음에 놓이는
현재 관측대상이다.

즉, ComplexTest는 독립적으로 갑자기 생긴 디렉토리가 아니라,
관측기준자리와 전체 흐름의 연속 이후에 놓이는 next observation target이다.


4. 전체 관계 요약

000_dot / 100_empty_position
= 관측기준자리
= 관측기준을 성립시키는 자리 조건

Structure_Principle/schema/101~121
+
SeungeFlow_Thinking/thinking_flow/
= 전체 흐름의 연속
= 구조 흐름과 사고 흐름의 이어짐

epluone/ComplexTest/
= 그 연속 흐름 다음에 놓인 현재 관측대상


5. gpt.github 최종 기준

gpt.github는 앞으로 GitHub 업로드와 Raw 주소 표시에서 다음 기준을 따른다.

- 000_dot과 100_empty_position은 관측기준자리 계열로 본다.
- 101~121과 thinking_flow는 전체 흐름의 연속 계열로 본다.
- epluone/ComplexTest는 그 연속 흐름 다음의 현재 관측대상으로 본다.
- Raw 주소는 AI가 문서 원문을 읽기 위한 우선 주소로 표시한다.
- GitHub view 주소는 사람이 위치를 확인하기 위한 주소로 표시한다.
