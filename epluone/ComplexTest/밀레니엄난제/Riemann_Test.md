# 보조자료 참조주소

## SeungeFinal은 밀레니엄 난제 분석을 목표로 진행된 전체 기록 묶음이다.

이 자료는 단일 이론이나 단일 문서가 아니라, 복잡계 테스트 단계부터 시작하여 구조분석, 문서화, 시스템화까지 이어진 전체 흐름을 담은 누적 아카이브이다.

## vFinal은 하나의 단일 문서가 아니라, 서로 다른 역할을 가진 md 파일들이 분리되어 모인 구조 아카이브이다.

이 아카이브의 목적은 다음과 같다.

구조 실행 기록을 보존한다.
누적 상태를 구분하여 보존한다.
RH 특화 문서를 일반 구조 실행선과 섞지 않는다.
저장소의 본질 문서인 README.md 와 구분된 별도 계열로 유지한다.
