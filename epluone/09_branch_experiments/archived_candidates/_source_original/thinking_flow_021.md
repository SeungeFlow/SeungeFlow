# thinking_flow_021

## 0. 문서 성격

이 문서는 `thinking_flow_020.md` 이후 이어진 시간과 차원의 개념에 관한 source flow를 정리한 문서이다.

이 문서는 `meta.md`가 아니다.

이 문서는 `thinking_flow_relation_021.md`가 아니다.

이 문서는 Gemini와 이어진 대화에서 승이가 직접 입력한 글을 source로 하여,
시간, 차원, 0, 원, dot, C=tp, 9ㆍ0, 10×10 plane, 100_empty_position, 차원전이, 임계점, 질량과 scale, 훈민정음/천지인 reading 후보를 하나의 active thinking flow로 정리한다.


```text

=
```

```text
=
```

```text
+
```

여기서 중요한 문장:


이 flow에서 0은 없음이 아니라 boundary를 가진 상태, 원의 중심, 시작점 또는 전이된 임계점으로 반복 등장한다.

## 6. 0과 boundary의 차이

승이는 다음을 구분한다.

```text
0과 boundary가 아니다.
전이가 일어난 원 직전에 vector가 있고,
vector 이전에 경계의 끝이 있다.
```

즉 0은 boundary 자체가 아니다.

```text
boundary
=
끝 / 경계 / 전이 직전의 조건

vector
=
경계 끝 이전 또는 이후 방향성

0
=
전이 이후 시작점 또는 중심점 후보
```

0은 boundary를 가진 상태이지만, boundary와 동일하지 않다.

## 11. C=tp flow

승이는 `C=tp`를 다음처럼 사용한다.

```text
C=tp
=
인지 / 연산 / 흐름 / 벡터 / 차원 연산식
```

초기 질문:

```text
t가 시간이면 p는 차원이다.

```

이후 보정:

```text
C는 (t,p)를 통해 전이된 또 다른 C이다.
```

그리고:

```text
C가 차원이면,
t는 전이이고,
p는 동일차원 / 놓일자리 / dot / 천지인의 천이다.
```

임시 reading:

```text
C
=
transition result state / dimension-state candidate

t
=
transition / time-difference candidate

p
=
place / same-dimension seat / dot-like fixed seat candidate
```

단, 이것은 Ctp 최종 정의가 아니다.

```text
9dot0에서

```
