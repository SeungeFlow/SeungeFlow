## 2. source 이동 원칙

```text
원본을 삭제하지 않는다.
압축파일은 가능하면 원형 그대로 보존한다.
압축해제본은 하위 디렉토리에 둔다.
source와 해석문을 같은 파일로 섞지 않는다.
AI 출력과 승이 입력을 가능하면 분리한다.
```
