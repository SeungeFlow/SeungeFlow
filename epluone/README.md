# epluone

## 역할

```text
Structure_Principle Formation Corpus 전체가 놓이는 장(field)이다. 하나의 문서가 아니라 형성장, 기반이론, 실행원리, 구조테스트, 구현 branch, root support, 응용 힌트를 분리 보존하는 Seed field다.
```

## 보존 규칙

```text
원문을 삭제하지 않는다.
하나의 문서로 합치지 않는다.
요약으로 닫지 않는다.
source와 interpretation을 분리한다.
이 디렉토리의 자료는 자기 자리를 유지한다.
```
